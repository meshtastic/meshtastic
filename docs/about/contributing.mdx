Meshtastic is built entirely by volunteers. The firmware, the mobile and web apps, the libraries, and the documentation all come from people contributing in their off hours, working on the things they care about. Whatever your skills, there is a place for you here.

## Contributing to Development

If you want to contribute to the development of Meshtastic, start with the [Developers](/docs/developers) section. It explains how the project works, how to get involved, and how to get your contributions accepted, and it links to guides for every part of the ecosystem: firmware, device APIs, the Android and Apple apps, the web client, the JavaScript and Python libraries, and the documentation.

You don't need to write code to make a difference. Testing pull requests, reporting issues, and improving documentation are some of the most valuable contributions we receive.

## Supporting Meshtastic Financially

The Meshtastic project benefits greatly from the contributions of our volunteers and the financial support from our community. As the project has evolved, the expansion of features and services has led to increased financial needs, including hardware for development, GitHub fees, hosting for public MQTT services, among others. This growth into a robust platform for decentralized off-grid mesh networking solutions showcases the dedication of our developers and the generosity of our financial supporters.

## Overview

Mesh Client is a cross-platform Electron desktop client for Meshtastic and MeshCore on macOS, Linux, and Windows with multi-language support, BLE, USB serial, Wi-Fi/TCP, MQTT, local SQLite history, and routing diagnostics.

## Features

- **Dual Protocol** — Full support for both Meshtastic and MeshCore simultaneously with a protocol switcher
- **Connectivity** — Bluetooth LE, USB Serial, WiFi/HTTP/TCP, MQTT
- **Radio & Module Config** — 18 region presets, 7 modem presets, channel URL import/export, MQTT with AES-128/256-CTR decryption, and module configuration (Telemetry, MQTT relay, Serial, Store & Forward, Range Test, Canned Messages, Remote Hardware/GPIO, Traffic Management, TAK)
- **PKI Security** — Key backup/restore, PKC remote node administration (firmware 2.5+)
- **Network Diagnostics** — Routing anomaly detection (hop_goblin, bad_route, route_flapping), channel utilization history, congestion attribution
- **Full-Featured Chat** — Durable outbox (SQLite), long message chunking, emoji reactions/tapbacks, link previews, message starring, @mention autocomplete, export chat
- **Node Management** — Node health scores, signal bars, distance filter, JSON export
- **Interactive Map** — Leaflet + OpenStreetMap with position trails and layer controls
- **Telemetry Charts** — Battery, SNR/RSSI, environment telemetry visualization
- **TAK Server** — Broadcast mesh node positions to ATAK/WinTAK/iTAK clients
- **Multi-Language** — 16 languages including English, Spanish, Ukrainian, German, Chinese Simplified, Portuguese Brazilian, French, Italian, Polish, Czech, Japanese, Russian, Dutch, Korean, Turkish, Indonesian
- **Log Panel** — Log analysis with in-app updates
- **Accessibility** — Keyboard navigation, screen reader support, reduce motion, Windows High Contrast

## Limitations

- Electron-based, so higher memory usage than CLI alternatives
- Bluetooth LE connection requires platform-specific support (not available on all Linux configurations)

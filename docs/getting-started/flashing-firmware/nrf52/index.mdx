## Choose a Flashing Method

nRF52, RP2040, and RP2350 devices are the easiest to update — no drivers or special software required. The device appears as a USB drive and you simply copy a file to it.

| Method | Best For | Difficulty |
|---|---|---|
| [**Drag & Drop**](/docs/getting-started/flashing-firmware/nrf52/drag-n-drop) (recommended) | Everyone — simplest method | Easy |
| [**Over-The-Air (OTA)**](/docs/getting-started/flashing-firmware/nrf52/ota) | Updating nRF52 devices wirelessly via Bluetooth | Moderate |

### Troubleshooting & Advanced Options

If you run into issues, these guides can help:

- [**Factory Erase**](/docs/getting-started/flashing-firmware/nrf52/nrf52-erase) — Clear stored data when upgrading between major firmware versions.
- [**Convert RAK4631-R**](/docs/getting-started/flashing-firmware/nrf52/convert-rak4631r) — Required if your RAK device shipped with the RUI3 bootloader instead of Arduino.
- [**Update Bootloader**](/docs/getting-started/flashing-firmware/nrf52/update-bootloader) — Fix issues caused by an outdated bootloader.
- [**SWDIO with Raspberry Pi**](/docs/getting-started/flashing-firmware/nrf52/swdio) — Flash via a direct debug connection when USB and Bluetooth aren't working.

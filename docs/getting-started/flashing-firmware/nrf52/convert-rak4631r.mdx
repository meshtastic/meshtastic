The RAK4631-R (RUI3) and RAK4631 (Arduino) are identical hardware — the only difference is the bootloader. Meshtastic requires the **Arduino bootloader**, so if your device shipped with RUI3, you need to convert it. This only needs to be done once.

## Method 1: USB DFU (Recommended)

1. Install [Python](https://www.python.org/downloads/) if you don't already have it.
2. Install adafruit-nrfutil:
   ```shell
   pip3 install adafruit-nrfutil
   ```
3. Download the bootloader: [WisCore_RAK4631_Board_Bootloader.zip](https://github.com/RAKWireless/WisBlock/releases/download/0.4.2/WisCore_RAK4631_Board_Bootloader.zip)
4. Connect your RAK device via USB.
5. Flash the bootloader (replace the serial port with the correct one for your system):
   ```shell
   adafruit-nrfutil --verbose dfu serial --package ./WisCore_RAK4631_Board_Bootloader.zip -p /dev/ttyACM0 -b 115200 --singlebank --touch 1200
   ```

   :::tip Finding your serial port
   - **Linux**: Usually `/dev/ttyACM0` or `/dev/ttyUSB0`
   - **macOS**: Usually `/dev/cu.usbmodem*` (use `ls /dev/cu.*` to find it)
   - **Windows**: Use Device Manager to find the COM port (e.g., `COM3`)
   :::

6. Once complete, continue with the normal [flashing instructions](/docs/getting-started/flashing-firmware/nrf52/drag-n-drop).

### If the DFU Flash Fails

If you see errors like `Timed out waiting for acknowledgement from device`, the RUI3 bootloader may need to be put into DFU mode manually. Follow the first section (through the `AT+BOOT` command) of the [RAK Wireless conversion guide](https://docs.rakwireless.com/product-categories/wisblock/rak4631-r/dfu/#converting-rak4631-r-to-rak4631), then retry step 5 above.

## Method 2: Hardware Debugger

This method requires a debug probe such as a [DAPLink](https://daplink.io/), [J-Link](https://www.segger.com/products/debug-probes/j-link/), or [RAKDAP1](https://msh.to/rakdap1/).

1. Install [Python](https://www.python.org/downloads/) if you don't already have it.
2. Install pyOCD:
   ```shell
   pip3 install pyocd
   ```
3. Download the bootloader: [WisCore_RAK4631_Board_Bootloader.hex](https://github.com/RAKWireless/WisBlock/releases/download/0.4.2/WisCore_RAK4631_Board_Bootloader.hex)
4. Connect the debug probe to your RAK device:
   [<img src="/img/rak4631-rakdap1.webp" style={{zoom:'25%'}} />](/img/rak4631-rakdap1.webp)
5. Flash the bootloader:
   ```shell
   pyocd flash -t nrf52840 .\WisCore_RAK4631_Board_Bootloader.hex
   ```
6. Continue with the normal [flashing instructions](/docs/getting-started/flashing-firmware/nrf52/drag-n-drop).

For other flashing methods, see the [RAK WisBlock bootloader repository](https://github.com/RAKWireless/WisBlock/tree/master/bootloader/RAK4630).

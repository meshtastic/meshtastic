A factory erase clears all stored configuration, logs, and data from your device's internal flash. This is useful when upgrading between major firmware versions, since the data format may change and cause issues like crash loops.

:::tip Web Flasher Alternative

You can also factory erase using the [Meshtastic Web Flasher](https://flasher.meshtastic.org): select your board, then click the **trash can icon** next to the Flash button.

:::

## nRF52 Devices

1. Download and unzip the latest firmware from [Meshtastic Downloads](https://meshtastic.org/downloads).
2. Connect your device to your computer with a USB data cable.
3. **Double-click the reset button** to enter bootloader mode.
4. A new USB drive will appear on your computer.
5. Open the drive and look at the file `INFO_UF2.TXT` — note the **SoftDevice** version number.
6. From the downloaded firmware, find the erase file that matches your SoftDevice version:
   - SoftDevice **7.x.x** → use `Meshtastic_nRF52_factory_erase_v3_S140_7.3.0.uf2`
   - SoftDevice **6.x.x** → use `Meshtastic_nRF52_factory_erase_v3_S140_6.1.0.uf2`
7. Copy the matching erase file onto the USB drive. The device will reboot.
8. To confirm the erase worked, install the [Meshtastic CLI](/docs/software/python/cli/) and run:
   ```shell
   meshtastic --noproto
   ```
   You should see the message: `Formatting... Done`.

## RP2040 Devices

1. Download [flash_nuke.uf2](https://datasheets.raspberrypi.com/soft/flash_nuke.uf2) to your computer.
2. **Hold the BOOTSEL button** and connect the device to your computer via USB while still holding.
3. The board will appear as a USB drive labeled **RPI-RP2**.
4. Copy `flash_nuke.uf2` to the drive.
5. The board will restart and the flash memory will be erased.

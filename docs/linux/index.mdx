**meshtasticd** is the Linux-native binary for running Meshtastic on devices with SPI or USB radios.

With `meshtasticd`, your Linux computer is transformed into a Meshtastic node, enabling it to send and receive messages, share location data, and interact with other Meshtastic devices over LoRa or UDP.

Features include:
- Lightweight! `meshtasticd` is built on the same core as the Meshtastic firmware.
- Supports SPI and USB radios.
- Packaged for most popular Linux distributions.
- Easy yaml-based configurations.
- Premade pinouts for popular [boards](hardware/boards).

Typical use cases:
- Always-on base stations
- Network gateways
- MQTT bridges
- Integration with home automation systems
- Remote monitoring nodes

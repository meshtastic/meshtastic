Meshtastic includes several built-in user interfaces (UIs), each designed to fit different device types, features, and preferences.

Use the links below to explore the documentation for each available UI.

## UIs

### [BaseUI](./baseui/)

The standard interface included with Meshtastic firmware. BaseUI provides straightforward navigation and access to key features directly on supported devices.

### [MeshtasticUI](./meshtasticui/)

Also called MUI, this standalone interface offers a modern, touch-friendly experience for compatible devices. It includes advanced tools for navigation, messaging, mapping, and configuration.

### [InkHUD](./inkhud/)

A power-efficient interface designed for devices with e-ink displays. InkHUD emphasizes clear visibility in all lighting conditions and a clean, simplified layout.

# Meshtastic Apple App Documentation

User and developer documentation for the Meshtastic iOS, iPadOS, macOS, watchOS, and visionOS app.

Use the sidebar navigation to browse the **User Guide** for app features and the **Developer Guide** for contributing to the project. Each guide opens on its own **What's New** page listing the last 12 months of changes.

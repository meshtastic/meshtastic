The Meteor Pro is a Meshtastic device from the Swiss vendor NomadStar, built on the RAK4631 (nRF52840) WisBlock core module with a Semtech SX1262 LoRa transceiver.

## Specifications

- **MCU**
  - nRF52840 (based on the RAK4631 WisBlock Core module)
    - Bluetooth BLE 5.0
- **LoRa Transceiver**
  - Semtech SX1262

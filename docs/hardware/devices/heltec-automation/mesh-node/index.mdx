## Mesh Node T096

<DeviceImage device="heltec-t096.svg" alt="Heltec Mesh Node T096" size="sm" center />

The Mesh Node T096 is a compact, ultra-low-power nRF52840 mesh node featuring an amplified SX1262 LoRa radio, a dual-band multi-constellation GNSS, and a 0.96-inch color TFT display. It supports Li-ion battery and solar charging, making it well suited as a low-power field or tracker node.

- **MCU:**
  - nRF52840 (Bluetooth 5 LE)
- **LoRa Transceiver:**
  - Semtech SX1262 with power amplifier (up to 28±1 dBm)
- **Frequency Options:**
  - 433 MHz
  - 470 - 510 MHz
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Navigation Module:**
  - UC6580 dual-band GNSS (GPS, GLONASS, BDS, Galileo, NAVIC, QZSS)
- **Connectors:**
  - USB-C
  - Li-ion battery and solar-panel charging

### Features

- Onboard 0.96-inch color TFT display.
- Amplified SX1262 LoRa radio (up to 28±1 dBm).
- Dual-band UC6580 GNSS.
- Li-ion battery with solar charging support.
- Ultra-low ~12 µA sleep current.

## Mesh Node T1

<DeviceImage device="heltec-meshnode-t1.svg" alt="Heltec Mesh Node T1" size="sm" center />

The Mesh Node T1 is a rugged, card-sized LoRa position tag built on the nRF52840 and SX1262, with multi-constellation GNSS, a 9-axis IMU, a buzzer, and a 0.96-inch TFT display. Its IP-rated enclosure and built-in battery make it a self-contained field and tracker node.

- **MCU:**
  - nRF52840 (Bluetooth 5 LE)
- **LoRa Transceiver:**
  - Semtech SX1262
- **Frequency Options:**
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Navigation Module:**
  - Multi-constellation GNSS (GPS, BeiDou, GLONASS, Galileo)
- **Battery Capacity:**
  - 1850 mAh built-in Li-ion
- **Connectors:**
  - Waterproof USB-C

### Features

- Onboard 0.96-inch TFT display.
- 9-axis IMU.
- Buzzer for notifications.
- IP67-rated enclosure.
- Card-sized form factor (85 x 55 x 9.5 mm); ~11 µA deep-sleep current.

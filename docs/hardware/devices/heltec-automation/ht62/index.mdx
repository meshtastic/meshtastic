The Heltec HT-CT62 is a tiny, low-cost surface-mount module that combines an ESP32-C3 MCU with an SX1262 LoRa radio. It is a bare module — with no display, battery, or enclosure — designed to be soldered onto a carrier board, making it a popular building block for DIY Meshtastic nodes.

## Specifications

- **MCU**
  - ESP32-C3FN4 (32-bit RISC-V, up to 160 MHz)
    - 4 MB SiP flash
    - 2.4GHz Wi-Fi & Bluetooth LE
- **LoRa Transceiver**
  - Semtech SX1262 (max TX 21±1 dBm, sensitivity to -134 dBm)
- **Frequency options**
  - 470 MHz
  - 868 MHz
  - 915 MHz
- **Connectors**
  - IPEX 1.0 (U.FL) antenna socket for LoRa
  - 1.27 mm castellated (stamp-hole) pins for SMT assembly

## Features

- Bare LoRa + MCU module — no display, GPS, buttons, buzzer, or battery on the module itself.
- Ultra-low ~10 µA deep-sleep current.
- Compact 17.78 x 17.78 x 2.8 mm SMT footprint for direct assembly onto a carrier board.

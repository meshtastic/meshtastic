The Super IO is the first IO board in the BASE System, transforming your Base board into a super capable portable device with standalone capabilities. Designed to stack with Base Uno or Base Duo, it adds GPS functionality, display support, and extended IO to expand your project possibilities.

## Specifications

- **GPS**
  - Built-in GPS module
- **Display**
  - Display connectivity support
- **Connectors**
  - Stacks directly onto Base Uno or Base Duo
  - Extended IO headers
- **Compatibility**
  - [Base Uno](../?base-system=base-uno)
  - [Base Duo](../?base-system=base-duo)

## Features

- GPS module for position tracking
- Display connectivity for visual feedback
- Extended IO capabilities for custom projects
- Stackable design maintains compact form factor
- Seamless integration with BASE System boards

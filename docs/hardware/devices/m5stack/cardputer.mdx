The Cardputer Mesh Kit bundles M5Stack's keyboard-equipped ESP32-S3 Cardputer ADV with a snap-on LoRa + GNSS cap ("Cap LoRa-1262"), preloaded with Meshtastic — a self-contained pocket mesh messaging terminal.

## Specifications

- **MCU**
  - ESP32-S3FN8 (Stamp-S3A module)
    - 2.4GHz Wi-Fi b/g/n, Bluetooth 5 LE
- **LoRa Transceiver**
  - Semtech SX1262 (Cap LoRa-1262 module; max TX +22 dBm, sensitivity -147 dBm)
- **Frequency options**
  - 868 - 923 MHz (covers EU 868 / US 915)
- **Navigation Module**
  - AT6668 GNSS (GPS, BeiDou, GLONASS, Galileo)
- **Connectors**
  - USB-C
  - EXT 2.54-14P expansion bus (for the LoRa cap)
  - HY2.0-4P Grove interface
  - microSD card slot

## Features

- 1.14-inch LCD (240 x 135).
- 56-key QWERTY keyboard.
- 1750 mAh rechargeable Li-ion battery.
- 6-axis IMU (BMI270).
- Audio: ES8311 codec, MEMS microphone, and 1 W speaker.
- Infrared (IR) emitter.
- Pre-installed with Meshtastic firmware.

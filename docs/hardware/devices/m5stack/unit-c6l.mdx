The M5Stack Unit C6L is a compact, enclosed LoRa communication unit built on the ESP32-C6 and SX1262. With an onboard OLED, buzzer, RGB LED, and button, it is a ready-to-use Meshtastic node.

## Specifications

- **MCU**
  - ESP32-C6 (dual RISC-V, 160 MHz + 20 MHz LP core)
    - 2.4GHz Wi-Fi 6, Bluetooth 5 LE, Zigbee 3.0, Thread 1.3
    - 16 MB flash
- **LoRa Transceiver**
  - Semtech SX1262 (max TX +22 dBm, sensitivity -147 dBm)
- **Frequency options**
  - 868 - 923 MHz (covers EU 868 / US 915)
- **Connectors**
  - USB-C (5V)
  - 2x SMA (one for 2.4 GHz Wi-Fi, one for LoRa)
  - HY2.0-4P Grove interface

## Features

- 0.66-inch SSD1306 monochrome OLED (64 x 48).
- Programmable RGB LED (WS2812C).
- Buzzer.
- User button.
- Compact 62 x 24 x 8 mm enclosure; ships with two SMA whip antennas.

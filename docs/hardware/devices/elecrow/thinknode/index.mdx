## ThinkNode M1

<DeviceImage device="thinknode_m1.svg" alt="Elecrow ThinkNode M1" size="lg" center />

The ThinkNode M1 is a compact Meshtastic device featuring an nRF52840 MCU, SX1262 LoRa transceiver, built-in GPS, and a 1.54" e-ink display. It includes a 1200 mAh rechargeable battery, and comes pre-installed with Meshtastic with an optional durable enclosure.

### Specifications

- **MCU:**
  - nRF52840
    - Bluetooth 5.0
- **LoRa Transceiver:**
  - Semtech SX1262
- **Frequency Options:**
  - US (902.0Mhz to 928.0Mhz)
  - EU_868
- **Navigation Module:**
  - L76K
- **Antenna:**
  - RP-SMA
- **Connectors:**
  - USB-C

### Features

- Meshtastic pre-installed.
- 1.54" e-ink display.
- 1200 mAh rechargeable battery.
- Integrated buzzer.
- Rotary encoder for backlight control.

## ThinkNode M3

<DeviceImage device="thinknode_m3.svg" alt="Elecrow ThinkNode M3" size="lg" center />

The ThinkNode M3 is a compact, card-style Meshtastic tracker featuring an nRF52840 MCU, Semtech LR1110 LoRa transceiver, and built-in GNSS module. It includes a 770 mAh rechargeable battery with magnetic charging, integrated environmental sensors, and an IP66-rated enclosure for indoor and outdoor positioning.

### Specifications

- **MCU**
  - nRF52840 (Bluetooth 5 LE)
- **LoRa Transceiver**
  - Semtech LR1110
- **Frequency options**
  - 915 MHz
  - 868 MHz
- **GNSS Module**
  - ATGM336H-5NR32
- **Battery Capacity**
  - Rechargeable lithium battery, 770 mAh
- **Charging**
  - USB magnetic charging cable

### Features

- Compact, pocket-sized design.
- Integrated Temperature, Humidity, and Accelerometer sensors.
- Four pogo pins for USB, DFU, serial logging, and charging.
- IP66 rated, waterproof and durable for various environments.

## ThinkNode M4

<DeviceImage device="thinknode_m4.svg" alt="Elecrow ThinkNode M4" size="lg" center />

The ThinkNode M4 is a power-bank-style Meshtastic node built around an nRF52840 MCU and a Semtech LR1110 transceiver. It pairs a 7000 mAh 18650 battery (with USB-C and wireless charging output) with multi-constellation GNSS tracking, so it doubles as a portable power bank and a mesh node.

### Specifications

- **MCU:**
  - nRF52840
    - Bluetooth 5.4 LE
- **LoRa Transceiver:**
  - Semtech LR1110
- **Frequency Options:**
  - US (902.0Mhz to 928.0Mhz)
  - EU_868
- **Navigation Module:**
  - Multi-constellation GNSS (GPS, BeiDou, GLONASS, QZSS)
- **Battery Capacity:**
  - 18650 Li-ion, 7000 mAh (25.2 Wh)
- **Connectors:**
  - USB-C (power in/out)
  - Qi wireless charging output

### Features

- Meshtastic pre-installed.
- Power-bank form factor with USB-C PD and wireless (Qi) charging output.
- Built-in GNSS tracker with track recording.
- Built-in ceramic GNSS antenna.

### Resources

- Firmware file: `firmware-thinknode_m4-X.X.X.xxxxxxx.uf2`

## ThinkNode M7

<DeviceImage device="thinknode_m7.svg" alt="Elecrow ThinkNode M7" size="lg" center />

The ThinkNode M7 is an indoor Meshtastic gateway built around an ESP32-S3 MCU and Semtech LR1110 transceiver. With wired Ethernet and Power-over-Ethernet (PoE), it provides plug-and-play fixed-infrastructure coverage.

### Specifications

- **MCU:**
  - ESP32-S3
    - 2.4GHz Wi-Fi b/g/n
    - Bluetooth 5 LE
- **LoRa Transceiver:**
  - Semtech LR1110
- **Frequency Options:**
  - US (902.0Mhz to 928.0Mhz)
  - EU_868
- **Network:**
  - 100 Mbps Ethernet (RJ45) with PoE
- **Connectors:**
  - USB-C
  - RJ45 Ethernet (PoE)

### Features

- PoE-powered, plug-and-play gateway.
- Wired Ethernet plus wireless connectivity for stable indoor coverage.
- Designed for indoor coverage and fixed regional gateway deployments.

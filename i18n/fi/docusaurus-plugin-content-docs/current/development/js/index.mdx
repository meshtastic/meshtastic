## Esittely

Meshtastic.js on JavaScript-kirjasto, joka tarjoaa käyttöliittymän Meshtastic laitteisiin. Sitä voidaan käyttää rakentamaan sovelluksia liittymään Meshtastic verkkoon. Tällä hetkellä tuetaan HTTP(S)- ja Bluetooth-yhteyksiä.

If you wish to view the code or contribute to development of the library, please visit the JavaScript code [GitHub page](https://github.com/meshtastic/web).

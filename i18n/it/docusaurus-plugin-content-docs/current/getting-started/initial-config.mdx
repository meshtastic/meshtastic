## Configura impostazioni regionali

Per iniziare a comunicare attraverso la mesh, è necessario impostare la tua regione. Questa impostazione controlla quale gamma di frequenza utilizza il dispositivo e dovrebbe essere impostata in base alla posizione regionale.

<TabItem value="android">

#### Android

1. Segui le istruzioni [installazione](/docs/software/android/installation) e [utilizzo](/docs/software/android/usage/) per [Meshtastic su Android](/docs/category/android-app).
2. Aprire l'app, connettersi al dispositivo dal telefono tramite USB Seriale o Bluetooth.
3. Una volta fatto l'abbinamento, clicca su "UNSET" accanto al nome del dispositivo.
4. Seleziona la regione dall'elenco in base alla tua posizione regionale.

</TabItem>
<TabItem value="apple">

#### Apple

:::info
Configurazione di Regione, Preimpostazione Modem e Limite Hop è disponibile su iOS, iPadOS e macOS a Impostazioni > Configurazione Radio > LoRa.
:::

</TabItem>
<TabItem value="cli">

#### CLI

1. Installa [Meshtastic PythonCLI](/docs/software/python/cli/installation)
   ```sh
   pip3 install --upgrade pytap2
   pip3 install --upgrade meshtastic
   ```
2. Esegui il seguente comando, sostituendo `<REGION-CODE>` con il codice della regione elencato sopra in base alla tua posizione regionale.
   ```sh
   meshtastic --set lora.region <REGION-CODE>
   ```

</TabItem>
<TabItem value="web">

#### Web

1. Apri l'interfaccia Web di Meshtastic: [client.meshtastic.org](https://client.meshtastic.org)
2. Vai al menu **LoRa**.
3. Sotto **Impostazioni regionali**, imposta la tua **Regione** in base alla tua posizione regionale.
4. Clicca **Salva**.

</TabItem>
</Tabs>

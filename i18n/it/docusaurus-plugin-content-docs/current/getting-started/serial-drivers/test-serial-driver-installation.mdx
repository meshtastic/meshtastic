### Prova Installazione Del Driver Seriale

È possibile verificare che si disponga di un cavo dati adeguato (piuttosto che di un cavo a sola ricarica) e che i driver appropriati per il sistema siano installati eseguendo il seguente test:

#### Linux

1. Collega il tuo dispositivo Meshtastic alla tua porta USB

2. Apri un **Terminal** e inserisci il seguente comando:

   ```shell
   lsusb
   ```

3. Dovresti vedere qualcosa come:

   ```shell
   ID xxxx:xxxx Silicon Labs CP210x UART Bridge
   # o
   ID xxxx:xxxx QinHeng Electronics USB Single Serial
   # o
   ID xxxx:xxxx Adafruit WisCore RAK4631 Board
   ```

</TabItem>
<TabItem value="macos">

#### macOS

1. Entra nella sezione `Menu Apple  > Informazioni su questo Mac > Maggiori informazioni > Report di sistema... > Hardware > USB`.
2. Dovresti vedere qualcosa di simile a una delle seguenti voci:

- `CP210X USB to UART Bridge Controller`
- `CH9102 USB to UART Bridge Controller`
- `WisCore RAK4631 Board`
- `USB Single Serial`

</TabItem>
<TabItem value="windows">

#### Windows

1. Entra in `Device Manager > Porte (COM & LPT)`
2. Dovresti vedere qualcosa di simile a una delle seguenti voci:

- `Silicon Labs CP210X USB to UART Bridge (COM5)`
- `Silicon Labs CH9102 USB to UART Bridge (COM5)`
- `USB-Enhanced-SERIAL CH9102 (COM5)`
- `USB Serial Device (COM5)`

</TabItem>
</Tabs>

:::info

Se non riesci a vedere il tuo dispositivo:

- Assicurati che il tuo cavo non sia solo per la ricarica, ma anche per il [trasferimento di dati](/docs/getting-started#verify-data-cable).
- È possibile che tu abbia bisogno di [reinstallare il driver seriale USB](/docs/getting-started/serial-drivers).

:::

## Installare driver USB-seriali per nRF52 / RP2040

:::caution

i dispositivi nRF52/RP2040 tipicamente non richiedono driver seriali. Usano il bootloader UF2 che fa apparire i dispositivi come unità flash. _NON_ scaricare i driver USB del dispositivo a meno che non sia necessario installare il supporto UF2.

:::

#### macOS

:::info

Con le ultime versioni di MacOS, il driver USB-Seriale è integrato. Se hai già scaricato/installato qualsiasi driver, rimuovili.

:::

##### Rimuovere il driver USB CH34x (macOS)

Se hai già scaricato/installato i driver macOS WCH-IC CH340/CH341
("CH341SER_MAC") tramite il CH34x_Install_V1.5.pkg, dovrai
Disinstallare l'estensione del kernel:

1. Scollega il tuo dispositivo
2. Apri il terminale ed esegui:
3. `sudo rm -rf /Library/Extensions/usbserial.kext`
4. Riavvia

##### Installa il driver CH34x

- [CH34x Driver- macOS Download](https://github.com/WCHSoftGroup/ch34xser_macos)

</TabItem>

#### Windows

:::important

Dopo aver installato il driver, assicurati di riavviare il computer per completare il processo di installazione.

Puoi anche [testare l'installazione del driver seriale](/docs/getting-started/serial-drivers/test-serial-driver-installation) in questo passaggio se necessario.

:::

### Flash del Firmware

Dopo aver installato i driver seriali, è ora possibile flashare il firmware Meshtastic sul dispositivo. Per procedere, seleziona il tipo di dispositivo appropriato per il tuo dispositivo.

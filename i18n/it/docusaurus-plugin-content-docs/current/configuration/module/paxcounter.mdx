Il modulo Paxcounter conta il numero di persone che passano da un'area specifica scansionando gli indirizzi WiFi e BLE MAC. È comunemente utilizzato nei negozi al dettaglio, nei musei e in altri spazi pubblici per monitorare il traffico pedonale e raccogliere dati preziosi per l'analisi.

Per utilizzare questo modulo, assicurati che i tuoi dispositivi abbiano la versione del firmware 2.2.17 o superiore.

:::info
Questo modulo può essere utilizzato solo con i dispositivi ESP32. Per utilizzare il modulo Paxcounter, è obbligatorio spegnere sia il WiFi che il Bluetooth nelle impostazioni di rete e Bluetooth.
:::

## Valori Di Configurazione Del Modulo Paxcounter

### Abilitato

Se il modulo è abilitato.

### Intervallo di Aggiornamento

L'intervallo in secondi di quanto spesso possiamo inviare un messaggio alla mesh quando viene rilevato un cambiamento di stato.

#### CLI

:::info

Tutte le opzioni di configurazione del modulo Paxcounter sono disponibili nella versione di python CLI 2.2.16 e successive.

:::

I comandi di esempio sono di seguito:

```shell title="Enable/Disable the Paxcounter Module"
meshtastic --set paxcounter.enabled true
meshtastic --set paxcounter.enabled false
```

```shell title="Set the Minimum Broadcast Interval to 900 seconds"
meshtastic --set paxcounter.paxcounter_update_interval 900
```

```shell title="Get the Paxcounter Module Configuration"
meshtastic --get paxcounter
```

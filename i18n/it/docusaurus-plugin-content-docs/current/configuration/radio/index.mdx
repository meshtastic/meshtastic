Ci sono diverse sezioni di configurazione nel firmware Meshtastic, questi sono suddivisi in modo che possano essere inviati come piccoli messaggi di amministrazione sulla mesh.

|      Nome                |                                                                             Descrizione                                                                                                                                                                    |
| :----------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| [Bluetooth](./bluetooth) |                                                   Le opzioni di configurazione bluetooth sono: Abilitata, Modalità di abbinamento e PIN fisso.                                                                                                             |
|  [Channels](./channels)  |                                                         Le opzioni di configurazione dei canali sono: indice, ruolo e impostazioni.                                                                                                                        |
|   [Device](./device)     |                                      Le opzioni di configurazione del dispositivo sono: Ruolo del dispositivo, uscita seriale, log di debug e ripristino di fabbrica.                                                                                      |
|  [Display](./display)    |                                     Le opzioni di configurazione dello schermo sono: Schermo sulla durata, Intervallo di Carosello automatico, Sempre Punta a Nord e Formato GPS.                                                                          |
|   [LoRa ](./lora)        | Le opzioni di configurazione LoRa sono: Regione, Preimpostazione del modem, Max Hop, Potenza di trasmissione, Larghezza di banda, Fattore di diffusione, Tasso di codifica, Offset di frequenza, Trasmissione disabilitata e Ignora l'Array in arrivo.     |
|  [Network](./network)    |                                                 Le opzioni di configurazione di rete sono: WiFi abilitato, WiFi SSID, WiFi PSK, modalità WiFi e server NTP.                                                                                                |
|  [Position](./position)  |             Le opzioni di configurazione della posizione sono: GPS Abilitato, Intervallo Aggiornamento GPS, Tempo Tentativo GPS, Posizione Fissa, Trasmissione Intelligente, Intervallo di Trasmissione e Flag Pacchetto Posizione.                        |
|   [Power](./power)       |   Le opzioni di configurazione di alimentazione sono: Corrente di carica, Risparmio energetico, Spegnimento dopo la perdita di energia, ADC Multiplier Override Intervallo di attesa Bluetooth, Intervallo di sonno leggero e Intervallo di Wake minimo.   |
|  [Security](./security)  |                        Le opzioni di configurazione di sicurezza sono: chiave pubblica, chiave privata, chiave di amministratore, è gestito, console seriale, log di debug e canale di amministrazione.                                                    |
|    [User](./user)        |                                                          Le opzioni di configurazione utente sono: nome lungo, nome corto e con licenza                                                                                                                    |

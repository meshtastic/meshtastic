Le opzioni di configurazione utente sono: nome lungo, nome breve, con patente e non messaggiabile. La configurazione utente utilizza un messaggio di amministrazione per inviare un protobuf `User`.

## Valori Di Configurazione Utente

### Nome Lungo

Un nome personalizzato per il tuo dispositivo.

Auto-generato per impostazione predefinita.

Se sei un radioamatore con patente e hai abilitato `IsLicensed`, questo dovrebbe essere impostato sul tuo callsign da radioamatore.

### Nome Breve

Un identificatore breve personalizzato per il tuo dispositivo.

Auto-generato per impostazione predefinita.

### Con patente (radioamatore)

Se sei un radioamatore con patente e hai considerato i [privilegi e restrizioni](/docs/faq/#amateur-radio-ham) di utilizzare Meshtastic con una patente da radioamatore, abilita questo flag.

Disabilitato di default.

Abilitando `IsLicensed`, dovresti anche rivedere le seguenti configurazioni correlate:

- **Utente:** `LongName` (Dovrebbe essere il tuo Callsign)
- **Canale:** `PSK` (dovrebbe essere vuoto, rimuovendo la crittografia)

### Non messaggabile

Indica se il nodo può essere messaggiato direttamente o meno

Il valore predefinito è Falso tranne per i ruoli che tipicamente non sarebbero monitorati per i messaggi diretti: ROUTER, ROUTER_LATE, REPEATER, SENSOR, LOST_AND_FOUND, TRACKER, TAK_TRACKER

#### CLI

:::info

Tutte le opzioni di configurazione utente sono disponibili nel CLI python. I comandi di esempio sono di seguito:

:::

Vedi le istruzioni per [Abilitare la patente da radioamatore](/docs/software/python/cli/usage#ham-radio-support)

:::tip

Poiché il dispositivo si riavvierà dopo che ogni comando viene inviato tramite CLI, si raccomanda quando si impostano più valori in una sezione di configurazione, che i comandi siano concatenati insieme in un unico comando.

```shell title="Example:"
meshtastic --set-owner 'il tuo nome nodo' --set-owner-short  'NODE'
```

:::

```shell title="Set the LongName Value"
meshtastic --set-owner 'il tuo nome nodo'
```

```shell title="Set the ShortName Value"
meshtastic --set-owner-short  'NODE'
```

```shell title="Enable Ham Mode, set name to license, disable encryption"
meshtastic  --set-ham 'CALLSIGN'
```

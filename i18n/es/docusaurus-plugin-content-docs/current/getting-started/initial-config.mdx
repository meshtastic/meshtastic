## Configuración regional

Para poder empezar a comunicarte sobre la malla, debes establecer tu región. Este ajuste controla qué rango de frecuencias utiliza tu dispositivo y debe configurarse de acuerdo con la región en donde te ubiques.

<TabItem value="android">

#### Android

1. Sigue las instrucciones de [instalación](/docs/software/android/installation) y de [uso](/docs/software/android/usage/) para [Meshtastic Android](/docs/category/android-app).
2. Abre la aplicación y conéctate al dispositivo desde tu teléfono por medio de USB o Bluetooth.
3. Una vez emparejado, haz clic en "UNSET", junto al nombre del dispositivo.
4. Selecciona la región de la lista según tu ubicación regional.

</TabItem>
<TabItem value="apple">

#### Apple

:::info
Los ajustes "Configuración de la Region", "Preajuste de Modem" y "Límite de Saltos" para iOS, iPadOS y macOS, están disponibles en Settiings > Radio Configuration > LoRa.
:::

</TabItem>
<TabItem value="cli">

#### CLI

1. Instala [Meshtastic PythonCLI](/docs/software/python/cli/installation)
   ```sh
   pip3 install --upgrade pytap2
   pip3 install --upgrade meshtastic
   ```
2. Ejecuta el siguiente comando, sustituyendo `<REGION-CODE>` con el código de región correspondiente a tu ubicación de los listados arriba.
   ```sh
   meshtastic --set lora.region <REGION-CODE>
   ```

</TabItem>
<TabItem value="web">

#### Web

1. Abre la interfaz web de Meshtastic: [client.meshtastic.org](https://client.meshtastic.org)
2. Navega al menú **LoRa**.
3. En **Configuración regional**, establece tu **Región** de acuerdo a tu ubicación regional.
4. Haz clic en **Save**.

</TabItem>
</Tabs>

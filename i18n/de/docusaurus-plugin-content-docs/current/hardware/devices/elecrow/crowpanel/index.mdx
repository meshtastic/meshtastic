Die CrowPanel Advance Serie wurde von Elecrow entworfen und ist eine vielseitige Reihe leistungsstarker HMI Touchanzeige in 2,4"; 2,8"; 3,5"; 4,3"; 5,0" und 7,0" Größen, alle basierent auf der ESP32-SE Plattform. Mit modularer drahtloser Unterstützung einschließlich LoRa über das Modul SX1262 wurden diese Anzeigen erfolgreich für den Einsatz mit Meshtastic angepasst. Die CrowPanel erweiterte Serie ist eine ideale Plattform für Meshtastic UI (MUI).

## Spezifikationen

- **MCU**
  - ESP32-S3
    - WLAN 2,4GHz b/g/n
    - Bluetooth 5.0
- **LoRa Sendeempfänger**
  - Semtech SX1262
- **Frequenzoptionen**
  - Alle Meshtastic Frequenzen.
- **Antenne**
  - U.FL/IPEX Antennenanschluss für LoRa
- **Anschlüsse**
  - USB-C (x2)
  - BAT (PH2.0-2P)
- **Anzeige**
  - IPS kapazitive Berührung
    - 2,4 Zoll (320x240)
    - 2,8 Zoll (320x240)
    - 3,5 Zoll (480x320)
    - 4,3 Zoll (800x480)
    - 5,0 Zoll (800x480)
    - 7,0 Zoll (800x480)

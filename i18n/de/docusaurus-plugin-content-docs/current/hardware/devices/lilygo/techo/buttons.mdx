## Funktionen

- **Kapazitive Touch Fläche (oben):**<br />
  - **Berühren:**
    - Bildschirm aktualisieren
    - _(im Bildschirmschoner)_ Wecken Sie den Bildschirm auf.
- **Reset Taste (Taste 1):**
  - **Single Press:** Power-on / Neustart.
  - **Doppelt drücken:** Aktivieren Sie den Bootloader Modus für die Firmware Aktualisierung.
- **Programm Taste (Taste 2):**
  - **Einfaches Drücken: **
    - Nächste Informationsseite anzeigen.
    - _(im Bildschirmschoner)_ Wecken Sie den Bildschirm auf.
    - _(wenn aus)_ Aktiviert den Bootloader Modus zur Firmware Aktualisierung.
  - **Doppeltes Drucken:** Senden Sie einen "Adhoc Ping": kündigen Sie das Gerät im Netzwerk an.
  - **3x drücken:** GPS aktivieren / deaktivieren.
  - **4x drücken:** Aktivieren / Deaktivieren Sie die Hintergrundbeleuchtung.
  - **Halten:** Herunterfahren.

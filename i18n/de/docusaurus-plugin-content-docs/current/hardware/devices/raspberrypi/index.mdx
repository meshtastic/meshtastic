### [Raspberry Pi Pico](./pico/)

Schnelle vielseitige Platine mit dem RP2040/RP2350.

| Name                         | MCU           | Funkchip  |     WLAN          |      BT           |  GPS  |
| :--------------------------- | :------------ | :-------- | :---------------: | :---------------: | :---: |
| [Raspberry Pi Pico](./pico/) | RP2040/RP2350 | SX1262    | 2,4 GHz b/g/n[^1] | nicht unterstützt | Nein  |

[^1]: 2.4GHz b/g/n WiFi Unterstützung ist auf der Raspberry Pi Pico **W** Variante verfügbar.

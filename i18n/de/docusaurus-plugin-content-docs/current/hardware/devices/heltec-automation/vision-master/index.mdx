## Vision Master E213

- **MCU:**
  - ESP32-S3R8 (WLAN & Bluetooth)
- **LoRa Sendeempfänger:**
  - HT-RA62(Semtech SX1262)
- **Frequenzoptionen:**
  - 433 MHz
  - 470 - 510 MHz
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Anschlüsse:**
  - USB-C
  - 2x1.25 Lithium Batterie Anschluss
  - 2 x SH1.0-4P Sensorschnittstelle
  - Antenne:
    - U.FL/IPEX Antennenanschluss für LoRa
    - Integrierter 2,4 GHz Platinenantenne, U.FL/IPEX Anschluss für 2,4 GHz externe Antenne.

### Merkmale

- Anzeigebildschirm mit 2,13 Zoll schwarz-weiß E-Ink
- Boot, Reset und Benutzertaste (GPIO 21)
- Kein GPS

### Programmierung

Um ESP32-S3 Geräte wie den Vision Master E213 zu programmieren, müssen Sie sie normalerweise den Espressifs Firmware Download Modus aktivieren. Benutzen Sie dazu den "1200bps Reset" Knopf im Web-Flasher um dies zu tun. Wenn diese Methode aus irgendeinem Grund nicht funktioniert, können Sie dem manuellen Prozess folgen.

:::warning

Fahren Sie nicht fort, es sei denn, eine Antenne ist angeschlossen, um mögliche Schäden am Funkgerät zu vermeiden.

:::

Der folgende Prozess wird das Gerät manuell im Espressif Firmware Download Modus aktivieren:

1. Gerät trennen.
2. Drücken und halten Sie die BOOT Taste.
3. Gerät anschließen.
4. Nach 2-3 Sekunden lassen Sie die BOOT Taste los.

Mit dem Gerät im Espressif Firmware Download Modus, können Sie mit der Programmierung durch eine der unterstützten Methoden fortfahren. Es wird generell empfohlen, den [Web Flasher](https://flasher.meshtastic.org/) zu benutzen. Wählen Sie "Heltec Vision Master E213" aus der Liste der Geräte.

### Anschlussplan

![HT-VME213_PIN_MAP](/img/hardware/heltec/HT-VME213_PIN_MAP.webp)

Bildquelle: [Heltec](https://resource.heltec.cn/download/HT-VME213/HT-VME213.png)

### Quellen

Auf der Platine befindet sich eine zusätzliche Schnittstelle für 2,4 GHz Funksignale. Beim Anschluss einer externen Antenne muss der Widerstand um 90 Grad nach links gedreht werden. Für weitere Details lesen Sie bitte die [FAQ auf der offiziellen Website von Heltec] (https://heltec.org/project/vision-master-e213/).

- Firmware Datei: `firmware-heltec-vision-master-e213-X.X.X.xxxxxxx.bin`
- Links zum Direktkauf:
  - [Heltec](https://msh.to/heltec-vision-master-e213/)
  - [AliExpress](https://msh.to/aliexpress-heltec-vision-master-e213/)

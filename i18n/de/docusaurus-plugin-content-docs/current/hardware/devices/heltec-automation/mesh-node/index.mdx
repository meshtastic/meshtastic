## Mesh Node T114

- **MCU:**
  - nRF52840 (Bluetooth)
- **LoRa Sendeempfänger:**
  - Semtech SX1262
- **Frequenzoptionen:**
  - 433 MHz
  - 470 - 510 MHz
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Anschlüsse:**
  - USB-C
  - 2x1.25 Lithium Batterie Anschluss
  - 2x1.25 Solarpanel Anschluss
  - 8x1.25 GNSS Anschluss
  - Antenne:
    - U.FL/IPEX Antennenanschluss für LoRa.
    - Integrierte 2,4 GHz Patchantenne.

### Merkmale

- Die 1,14 Zoll TFT LCD Anzeige ist optional.
- GPS-Modul ist optional.
- Ausgezeichneter niedriger Energieverbrauch.

### Programmierung

1. DFU Modus manuell durch Doppelklick auf den RST Knopf aktivieren.

2. Kopieren Sie die vorbereitete Firmware UF2 Datei auf das Gerät.

## V3/V3.1

::::info
This device may have issues charging a connected battery if utilizing a USB-C to USB-C cable. It's recommended to use a USB-A to USB-C cable.

:::caution
Seien Sie vorsichtig, wenn Sie den USB-C-Anschluss benutzen. Dieses Gerät hat keinen ESD-Schutz für den CP2102 USB zu seriellen UART Schaltkreis.
:::

::::

- **MCU:**
  - ESP32-S3FN8 (WLAN & Bluetooth)
- **LoRa Sendeempfänger:**
  - Semtech SX1262
- **Frequenzoptionen:**
  - 433 MHz
  - 470 - 510 MHz
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Anschlüsse:**
  - USB-C
  - Antenne:
    - Separate 2,4 GHz Spiralantenne für WiFi/Bluetooth
    - U.FL/IPEX Antennenanschluss für LoRa

### V3.1 Unterschiede

Firmware bleibt die gleiche wie V3. Vergleichen Sie die Schaltpläne: [V3.0](https://web.archive.org/web/20221127054612/https://resource.heltec.cn/download/WiFi_LoRa32_V3/HTIT-WB32LA(F)_V3_Schematic_Diagram.pdf) und [V3.1](https://resource.heltec.cn/download/WiFi_LoRa_32_V3/HTIT-WB32LA(F)_V3.1_Schematic_Diagram.pdf). Wesentliche Unterschiede:

- Entfernung des Schaltkreises [FDG6322C](https://www.mouser.com/datasheet/2/308/1/FDG6322C_D-2312203.pdf) (Doppel-N & P-Kanal FET) aus der V3.1 Stromversorgung.
- Antennenfilterwerte in V3.1 (L11 = 1.8 nH, C15 = 2.7 nH, C24 = 1.8 pF) orientiert sich stärker an ESP32-S3 Referenzdesign als V3.0 (L11 = 1.6 nH, C15 = 6.9 pF, C24 = 2.4 pF).

### Merkmale

- 0,96 Zoll OLED Anzeige eingebaut
- Benutzer- und Reset-Tasten
- Kein GPS

### Meshtastic I2C Anschlüsse

- SDA: GPIO41
- SCL: GPIO42

## Wireless-Tracker v1.0

:::

- **MCU:**

- **LoRa Sendeempfänger:**
  - Semtech SX1262
- **Frequenzoptionen:**
  - 470 - 510 MHz
  - 863 - 870 MHz
  - 902 - 928 MHz
- **Anschlüsse:**
  - USB-C
  - Antenne:
    - Separate 2,4 GHz Spiralantenne für WiFi/Bluetooth
    - U.FL/IPEX Antennenanschluss für LoRa und GPS

### Merkmale

- 0,96 Zoll OLED Anzeige eingebaut
- Benutzer- und Reset-Tasten

### Programmierung

Um ESP32-S3 Geräte wie den Wireless-Tracker zu programmieren, müssen Sie sie normalerweise den Espressifs Firmware Download Modus aktivieren. Benutzen Sie dazu den "1200bps Reset" Knopf im Web-Flasher um dies zu tun. Wenn diese Methode aus irgendeinem Grund nicht funktioniert, können Sie dem manuellen Prozess folgen.

:::warning

Fahren Sie nicht fort, es sei denn, eine Antenne ist angeschlossen, um mögliche Schäden am Funkgerät zu vermeiden.

:::

Der folgende Prozess wird das Gerät manuell im Espressif Firmware Download Modus aktivieren:

1. Gerät trennen.
2. Drücken und halten Sie die BENUTZER Taste.
3. Gerät anschließen.
4. Nach 2-3 Sekunden lassen Sie die BENUTZER Taste los.

Mit dem Gerät im Espressif Firmware Download Modus, können Sie mit der Programmierung durch eine der unterstützten Methoden fortfahren. Es wird generell empfohlen, den [Web Flasher](https://flasher.meshtastic.org/) zu benutzen. Wählen Sie "Heltec Wireless Tracker" aus der Liste der Geräte.

![HT-Tracker_V1 Pin Map](/img/hardware/HT-Tracker_V1_Pin_Map.webp)

Bildquelle: [Heltec](https://heltec.org/project/wireless-tracker/)

SenseCAP T1000-E ist ein leistungsstarker Tracker für Meshtastic. So klein wie eine Kreditkarte, passt mühelos in die Tasche oder kann an Ihrer Kleidung befestigt werden. Er integriert Semtechs LR1110, Nordics nRF52840 und das AG3335 GPS Modul von Mediatek und bietet Meshtastic Benutzern eine hochpräzise, energiesparende Standort- und Kommunikationslösung.

:::warning[Note]
Derzeit können LR1110 Funkgeräte keine Meshtastic Pakete von den älteren SX127x Funkgeräten empfangen, um dies zu beheben. Dies erfordert eine aufwändige Änderung. Die Übertragung funktioniert und wenn Sie durch ein SX126x Funkgerät weitergeleitet werden, dann können Sie dennoch Pakete von SX127x Funkgerät empfangen.
:::

### Spezifikationen

- **MCU**
  - Nordic nRF52840 (unterstützt Bluetooth)
- **LoRa Sendeempfänger**
  - Semtech LR1110
- **Frequenzoptionen**
  - 865-928 MHz
- **Navigationsmodul**
  - Mediatek AG3335 GPS chip
- **Akkukapazität**
  - Lithium Akkue, 700mAh, wiederaufladbar
- **Ladung**
  - magnetisches USB Ladekabel

### Merkmale

- **Taste und Summer**: Für Benutzerinteraktion und Warnungen.
- **Pogo Pins**: Vier Pins für USB, DFU, serielle Protokollierung und Laden.
- **Gehärtetes Gehäuse**: IP65 Rate, wasserdicht und haltbar für unterschiedliche Umgebungen.

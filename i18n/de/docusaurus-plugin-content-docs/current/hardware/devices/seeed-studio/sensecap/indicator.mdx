Ein 4-Zoll-Touchscreen-Gerät mit doppelter MCU, dem ESP32 und RP2040, das WLAN, BLE und LoRa®️ unterstützt.

### Spezifikationen

- **MCU**
  - ESP32S3 (WLAN & Bluetooth) + RP2040
- **LoRa Sendeempfänger**
  - Semtech SX1262
- **Frequenzoptionen**
  - 862-930 MHz
- **Anschlüsse**
  - 2 USB-Port Typ C
  - 2 Grove Connectors: GPS und 400+ Grove Sensoren
  - Antennenanschluss Buchse

### Merkmale

- **Anzeige**: 3,95 Zoll, kapazitiv RGB Touch.
- **2 Grove Anschlüsse**: 400+ Grove kompatible GPIOs für flexible Erweiterungsoptionen.

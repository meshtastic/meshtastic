## Seeed Wio-WM1110 Entwicklungskit

:::note Externes GPS erforderlich
Die LR1110 GNSS Funktionalität funktioniert noch nicht. Seeed empfiehlt das Grove - GPS (Air530).
:::

- **MCU**
  - Nordic nRF52840
- **LoRa Sendeempfänger**
  - Semtech LR1110
- **Frequenzoptionen**
  - 868 MHz
  - 915 MHz
  - 923 MHz
- **Navigationsmodul**
  - Grove GPS Air530 (unterstützt GPS, Beidou, Glonass, Galileo, QZSS, SBAS)
- **Anschlüsse**
  - USB-C
  - LoRa Antenne: SMA Antenne und U.FL/IPEX
  - GNSS Antenne: RP-SMA Antenne und U.FL/IPEX
  - NFC Antenne: U.FL/IPEX
  - GPIO
  - I2C x1
  - UART x1
  - Solarpanel
  - SWDIO

### Merkmale

- Temperatur und Feuchtigkeitssensor (SHT41)
- 3-Achsen Beschleunigungssensor (LIS3DHTR)
- Reset Taster, Power Brücken, 2 Benutzertasten
- AAA Batterie x3
- Bildschirm wird separat verkauft

### Quellen

- Firmware Datei: `firmware-wio-sdk-wm1110-X.X.X.xxxxxxx.bin`
- Links zum Direktkauf:
  - International
    - [Seeed Studio](https://www.seeedstudio.com/Wio-WM1110-Dev-Kit-p-5677.html)

## Wio Tracker 1110 Dev Kit für Meshtastic

- **MCU**
  - Nordic nRF52840 (BLE 5.3)
- **LoRa Sendeempfänger**
  - Semtech LR1110
- **Frequenzoptionen**
  - 868 MHz
  - 915 MHz
  - 923 MHz
- **Navigationsmodul**
  - Semtech LR1110
- **Anschlüsse**
  - USB-C

  - GNSS Antenne: auf der Platine und U.FL/IPEX

### Merkmale

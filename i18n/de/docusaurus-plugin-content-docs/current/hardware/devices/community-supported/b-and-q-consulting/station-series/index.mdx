Die Station Serie von Geräten, die von Neil Hao von B&Q Consulting und betrieben mit Meshtastic, sind kompakte und langlebige LoRa Geräte, die für eine leistungsstarke Funkkommunikation entwickelt wurden. Sie verfügen über eine leistungsstarken Verstärker für LoRa Kommunikation, einen robusten SMA-Antennenanschluss und eine Vielzahl externer IO Schnittstellen.

Für umfassendere Informationen über die Station Serie von Geräten besuchen Sie bitte die B&Q Consulting Station Serie [Wiki Seite.](https://wiki.uniteng.com/en/meshtastic/station-g2)

## Spezifikationen

- **MCU**
  - ESP32 WROOM
    - WLAN
    - Bluetooth 4.2
- **LoRa Sendeempfänger**
  - Semtech SX1262
  - Zusätzlicher 35 dBm LoRa Leistungsverstärker
- **Frequenzoptionen**
  - US-915 MHz
  - EU-868 MHz
- **Navigationsmodul**
  - Optional
- **Antenne**
  - SMA Buchse
- **Anschlüsse**
  - USB-C (PD Protokoll)

## Merkmale

- Meshtastic vorinstalliert
- Benutzer Taste
- 1,3 Zoll OLED Anzeige
- Optionales GPS-Modul und IO-Erweiterungssockel
- Optionale [12V Batterie Dockingstation](https://shop.uniteng.com/product/12v-battery-docker-for-station-edition-g1/), die als zusätzliche Stromversorgung verwendet werden kann, oder in Szenarien, die Mobilität erfordern

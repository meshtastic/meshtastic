## Lora v1

:::warning
Nicht empfohlen mit einem Akku! Diese Platinen enthalten die falschen Komponenten in der LiPo Akku Ladeschaltung, sodass der Akku überladen werden kann.
:::

:::caution
Diese Platine ist noch in der Produktion, wird aber aus verschiedenen Gründen nicht für Neuankäufe oder für unbeaufsichtigte Installationen empfohlen. Die Firmware Unterstützung ist läuft aus. Im Zweifelsfall wählen Sie die Lora T3S3 Platine.
:::

- **MCU**
  - ESP32 (WLAN & Bluetooth)
- **LoRa Sendeempfänger**
  - Semtech SX1276
- **Frequenzoptionen**
  - 915 MHz
  - 868 MHz
- **Anschlüsse**
  - Micro USB
  - Antenne: U.FL Antennenanschluss

### Merkmale

- 0,96 Zoll OLED Anzeige eingebaut

### Quellen

- Firmware Datei: `firmware-tlora-v1-X.X.X.xxxxxxx.bin`

## Lora v2.1-1.6

:::caution
Frühe Versionen dieser Platinen enthalten die falschen Komponenten in der LiPo Akku Ladeschaltung, sodass der Akku überladen werden kann. Die nach 2021 gekauften Platinen sollten dieses Problem nicht haben, es sei denn, die Version ist T3_v1.6 20180606.
:::

:::caution
Diese Platine ist noch in der Produktion, wird aber aus verschiedenen Gründen nicht für Neuankäufe oder für unbeaufsichtigte Installationen empfohlen. Die Firmware Unterstützung ist läuft aus. Im Zweifelsfall wählen Sie die Lora T3S3 Platine.
:::

- **MCU**
  - ESP32 (WLAN & Bluetooth)
- **LoRa Sendeempfänger**
  - Semtech SX127x
- **Frequenzoptionen**
  - 433 MHz
  - 868 MHz
  - 915 MHz
  - Eine Variante dieser Platine hat einen temperaturgeregelten Oszillator (TCXO) für eine verbesserte Frequenzstabilität.
- **Anschlüsse**
  - Micro USB
  - Antennen: SMA-Antennenanschluss

### Variationen der Identifizierung von Marken

- FCC ID: 2ASYE-T3-V1-6-1
- "T3_v1.6.1 20210104" auf der Platine
- "T3_v1.6 20180606" auf der Platine
- "Model T3 V1.6.1" auf dem FCC Aufkleber

### Merkmale

- 0,96 Zoll OLED Anzeige eingebaut
- Energie und Reset Taste
- microSD-Anschluss
- Kein GPS

### Quellen

- Firmware Datei: `firmware-tlora-v2-1-1.6-X.X.X.xxxxxxx.bin`

<TabItem value="v3.0">

## Lora v3.0

:::caution
Dieses Board verwendet einen Semtech Chip der ersten Generation und wird daher nicht für neue Käufer empfohlen. Die Firmware Unterstützung ist läuft aus. Im Zweifelsfall wählen Sie die Lora T3S3 Platine.
:::

- **MCU**
  - ESP32 (WLAN & Bluetooth)
- **LoRa Sendeempfänger**
  - Semtech SX127x mit TCXO für verbesserte Frequenzstabilität
- **Frequenzoptionen**
  - 433 MHz
  - 868 MHz
  - 915 MHz
- **Anschlüsse**
  - USB-C
  - Antenne: IPEX-Antennenanschluss

### Merkmale

- Eingebaut in OLED 0,96 Zoll oder 2,13 Zoll E-Ink Anzeige
- Energie- und Reset Schalter, Benutzertaste
- microSD-Anschluss
- Solarpaneel und Batterieanschluss, enthält Solar-Ladeschaltung (CN3065)
- QUIIC jeweils für UART und I2C
- Kein GPS

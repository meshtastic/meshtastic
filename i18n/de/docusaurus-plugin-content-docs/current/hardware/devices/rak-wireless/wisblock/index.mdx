Der RAK WisBlock ist ein modulares Hardwaresystem, das zum Bau von Meshtastic Geräten verwendet wird.

RAKwireless verkauft mehrere **Meshtastic Starter Kits** die das Minimum enthalten, das Sie benötigen, um sofort loszulegen. Es stehen zwei Optionen zur Verfügung (mit jeder Variante zur Anpassung), ein [nRF52 Kernmodul](https://msh.to/rak4631) und ein mit einem [RP2040 Kernmodule](https://msh.to/rak11310.

Wenn Sie Teile separat kaufen möchten, benötigen Sie eine [WisBlock Basisplatine](./base-board) und ein [WisBlock Kernmodul](./core-module). Bitte stellen Sie sicher, dass Sie beim Kauf die korrekte Betriebsfrequenz für Ihr Land wählen.

Optional können Sie Zubehör wie ein GPS Modul, Bildschirm, Sensoren oder andere Module erwerben.

Eine vollständige Liste der RAKwireless Meshtastic-kompatiblen Geräte und Komponenten finden Sie in der [Meshtastic Kollektion](https://msh.to/rak-collection).

Bitte lesen Sie in der RAK Dokumentation nach, wie Sie Ihre Hardware mit der Basisplatine verbinden können, um sicherzustellen, dass Sie das Gerät nicht beschädigen.

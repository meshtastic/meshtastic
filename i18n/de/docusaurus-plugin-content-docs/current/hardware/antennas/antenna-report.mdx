## Gemeinschaftsbewertungen

Das Archiv der Meshtastic Antennenberichte wurde von [RicInNewMexico](https://github.com/RicInNewMexico) als Projekt gestartet, bevor es offiziell an die Meshtastic Organisation übertragen wurde. Es enthält Testberichte und Auswertungen verschiedener Antennen, die häufig mit Meshtastic Geräten verwendet werden, die von Mitgliedern der Gemeinschaft beigetragen wurden.

Diese Benutzer haben die Mühe gemacht zu testen und ihre Meinung darüber abzugeben, ob bestimmte Antennen optimal mit Meshtastic Geräten funktionieren.

Wenn Sie einen Testbericht oder eine Bewertung beitragen möchten, dann lesen Sie bitte die Richtlinien im Github Repository der [Meshtastic Antennen Berichte](https://github.com/meshtastic/antenna-reports).

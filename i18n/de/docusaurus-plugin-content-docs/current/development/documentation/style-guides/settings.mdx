## Übersicht

Die Seiten für Einstellungen sollte sich ausschließlich auf die Einstellungen und die Konfiguration des Gerätes beschränken. Hardware ist oft ein verwandtes Thema, aber diese Seiten sollten nicht versuchen, das Anschließen von Zubehör erklären. Erwähnen Sie, dass Zubehör benötigt wird, und verlinken Sie auf die entsprechenden Seiten.

## Darstellung

### Übersicht

Dieser Abschnitt sollte die Gruppe der Einstellungen beschreiben und was sie für das Gerät tun.

### Einstellungen

Dieser Abschnitt beginnt mit einer alphabetisierten Tabelle von Einstellungen, verfügbaren Werten und Standardwerten.

Verfügbare Werte sollten mit den Markierungen \`\` um den Wert aufgelistet werden.

| Art des Werts |      Beispiel               |
| :-----------: | :-------------------------: |
|   Boolean     |    `true`, `false`          |
|    Liste      | `Apfel`, `Banane`, `Orange` |
|   Bereich     |     `0` - `100`             |

Nach der Tabelle wird jede Einstellung im Detail beschrieben. Wenn die verfügbaren Optionen für eine Einstellung zusätzliche Erläuterungen benötigen, sollte eine Tabelle verwendet werden, um jede verfügbare Option zu beschreiben.

### Details

Wenn weitere Details benötigt werden, kann dieser optionale Abschnitt diese erklären. Dazu gehören Voraussetzungen, Links zu Hardware-Anleitungen usw.

### Beispiele

Dieser optionale Abschnitt kann Beispiele für Konfigurationen enthalten, bei denen mehrere Einstellungen für einen bestimmten Anwendungsfall eingerichtet werden müssen.

#### Anwendung


### Code-Blöcke

#### Anwendung

Geben Sie immer die Sprache an, die direkt nach dem Start des Code-Blocks verwendet wird (\`\`\`).

:::note
Für Kommandozeilenbeispiele verwenden Sie bitte `shell` und keine der anderen Variante.
:::

Für weitere Informationen lesen Sie bitte die relevante [Docusaurus Seite](https://docusaurus.io/docs/markdown-features/code-blocks)

````
```ts title="Demo"
export const typedArrayToBuffer = (array: Uint8Array): ArrayBuffer => {
  return array.buffer.slice(
    array.byteOffset,
    array.byteLength + array.byteOffset
  );
};
```
````

## Wie man beitragen kann

Übersetzungen in die Meshtastic Documentation einzubringen hilft, das Projekt einem breiteren Publikum zugänglich zu machen. Folgen Sie diesen Schritten, um Ihre Übersetzungen über Crowdin hinzuzufügen:

1. **Zugriff auf Crowdin**: Besuchen Sie die Crowdin Seite des Meshtastic Projekts unter [https://crowdin.meshtastic.org](https://crowdin.meshtastic.org).
2. **Erstellen Sie ein Konto**: Klicken Sie auf 'Anmelden' in der oberen rechten Ecke der Seite und folgen Sie den Anfragen, um ein Crowdin-Konto zu erstellen.
3. **Navigieren Sie zum Projekt**: Sobald Sie eingeloggt sind, suchen und wählen Sie das Projekt 'Docs' aus dem Crowdin-Dashboard.
4. **Wählen Sie eine Sprache**: Finde Sie die Sprache, für die Sie Übersetzungen beitragen möchten, und wähle sie diese aus. Eine Liste der Dateien, aus denen sich die Website der Meshtastic Dokumentation zusammensetzt, wird angezeigt. Wählen Sie die zu übersetzende Datei aus und klicken Sie dann oben auf die Schaltfläche 'Zum Editor', um mit der Übersetzung zu beginnen.
5. **Start der Übersetzung**: Im Editor sehen Sie die Datei, die Sie auf der linken Seite übersetzen. Artikel, die übersetzt werden können, typischerweise nach Sätzen aufgeschlüsselt, werden rot hervorgehoben. Klicken Sie auf ein Element, um es auszuwählen. Die Markierungsfarbe wird gelb und zeigt an, dass es ausgewählt ist. Geben Sie dann Ihre Übersetzung in das Editorfeld rechts ein. Wenn Sie mit einem Artikel (Satz) fertig sind, klicken Sie auf 'Speichern', um Ihre Übersetzung zu speichern. Wiederholen Sie diesen Vorgang für jedes Element, das Sie übersetzen möchten.

Ihr Beitrag wird überprüft und nach Genehmigung wird Ihre Übersetzung in die Meshtastic Dokumentation aufgenommen. Vielen Dank, dass Sie dabei geholfen haben, die Reichweite von Meshtastic zu vergrößern!

## Einleitung

Meshtastic.js ist eine JavaScript-Bibliothek, die eine Schnittstelle zu Meshtastic Geräten zur Verfügung stellt. Sie kann verwendet werden, um Anwendungen zu erstellen, die mit einem Meshtastic Netzwerk verbinden. Derzeit werden HTTP(S) und Bluetooth-Verbindungen unterstützt.

Wenn Sie sich den Code ansehen oder zur Entwicklung der Bibliothek beitragen möchten, besuchen Sie bitte die JavaScript Quellcode [Seite auf Github](https://github.com/meshtastic/web).

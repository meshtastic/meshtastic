## Vorwort

Das von der Bibliothek verwendete Ereignis-System wird von RxJS bereitgestellt. Dies ist die grundlegende Art und Weise, wie Sie mit Geräten und dem Meshtastic Netzwerk interagieren.

:::tip
Vollständige Anleitung zur Verwendung von RxJS ist verfügbar unter [rxjs.dev](https://rxjs.dev/guide/overview)
:::

...
:::info
Viele der Nachrichtentypen, die das Gerät senden soll, werden automatisch verarbeitet und dekodiert für Sie ihren eigenen Ereignisverlauf.
:::

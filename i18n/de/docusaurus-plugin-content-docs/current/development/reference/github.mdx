## Übersicht

Das Meshtastic Projekt wird auf GitHub gehostet, einer beliebten Web-basierten Plattform für Versionskontrolle und Softwareentwicklung. GitHub ermöglicht Entwicklern Änderungen an ihrem Code zu verwalten und zu verfolgen, mit anderen zusammenzuarbeiten und ihre Arbeit zu verteilen.

Wenn Sie zum Meshtastic Projekt beitragen oder einfach auf dem Laufenden bleiben möchten, dann benötigen Sie ein GitHub Konto.

## GitHub Konto einrichten

1. Gehen Sie zu [github.com](https://github.com)
2. Klicken Sie auf "Anmelden" in der oberen rechten Ecke.
3. Folgen Sie den Anweisungen, um Ihr Konto zu erstellen.

Mit einem GitHub Account können Sie:

- Zugriff auf die Meshtastic Projektarchive.
- Melden Sie Probleme oder Fehler, die Sie finden.
- Änderungen oder neue Funktionen durch Pull-Requests vorschlagen.
- Diskutieren und arbeiten Sie mit anderen Mitwirkenden zusammen.

## Meshtastic auf GitHub

Das Meshtastic Projekt wird unter [Meshtastic GitHub](https://github.com/meshtastic) organisiert. Hier sind einige der wichtigsten Repositories:

- [Firmware](https://github.com/meshtastic/firmware): Der Firmware Quellcode für Meshtastic Geräte.
- [Meshtastic-Android](https://github.com/meshtastic/Meshtastic-Android): Die Android-Anwendung für Meshtastic.
- [Web](https://github.com/meshtastic/web): Der Meshtastic Web Client.
- [Protocol Buffer](https://github.com/meshtastic/protobufs): Die Protocol Buffer Definitionen die von Meshtastic verwendet werden.
- [Meshtastic-Apple](https://github.com/meshtastic/Meshtastic-Apple): Apple iOS, iPadOS & macOS Clients für Meshtastic.
- [meshtastic](https://github.com/meshtastic/meshtastic): Die Meshtastic Projekt-Website und Dokumentation.
- [python](https://github.com/meshtastic/python): Die Python Kommandozeile und die API zur Kommunikation mit Meshtastic Geräten.

## Mitmachen bei Meshtastic

Wenn Sie an einem Beitrag zum Meshtastic Projekt interessiert sind, müssen Sie sich an die Richtlinien halten, die in den Projektarchiven des Projekts beschrieben werden. Diese Richtlinien beinhalten in der Regel Anweisungen, wie Sie Ihre Entwicklungsumgebung einrichten, Programmierkonventionen und den Prozess zum Einreichen von Pull-Requests erstellen.

Zusätzlich enthalten viele Meshtastic Repositories eine `README.md` Datei, die einen Überblick über das Projekt, die Installationsanweisungen und andere relevante Informationen bietet.

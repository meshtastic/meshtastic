Die folgenden Bibliotheken müssen installiert sein, bevor `meshtasticd` gebaut wird:

<Tabs groupId="operating-systems" queryString="os">
<TabItem value="debian" label={<><Icon icon="mdi:debian" style={{ marginRight: "0.25rem" }} height="1.5rem" /> Debian</>} default>
```shell
# Basisabhängigkeiten
sudo apt install libgpiod-dev libyaml-cpp-dev libbluetooth-dev libusb-1.0-0-dev libi2c-dev libuv1-dev libxkbcommon-dev libinput-dev
# Optionale Abhängigkeiten für Webserver-Unterstützung
sudo apt install openssl libssl-dev libulfius-dev liborcania-dev
# Optionale Abhängigkeiten für SDL-Unterstützung
sudo apt install libsdl2-dev
```
</TabItem>

<TabItem value="fedora" label={<><Icon icon="mdi:fedora" style={{ marginRight: "0.25rem" }} height="1.5rem" /> Fedora</>}>
```shell
# Aktivieren Sie das COPR-Repository der Meshtastic Build Tools.
sudo dnf copr enable @meshtastic/build-tools
# Basisabhängigkeiten
sudo dnf install gcc-c++ pkgconfig(yaml-cpp) pkgconfig(libgpiod) pkgconfig(bluez) pkgconfig(libusb-1.0) libi2c-devel pkgconfig(libuv)
# Optionale Abhängigkeiten für Webserver-Unterstützung
sudo dnf install pkgconfig(openssl) pkgconfig(liborcania) pkgconfig(libyder) pkgconfig(libulfius)
# Optionale Abhängigkeiten für SDL-Unterstützung
sudo dnf install sdl2-kompatibel-dev
```
</TabItem>
</Tabs>

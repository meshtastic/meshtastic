Das Besucherzählermodul zählt die Anzahl der Personen, die einen bestimmten Bereich passieren, indem sie nach WLAN und BLE MAC Adressen scannen. Es wird häufig in Einzelhandelsgeschäften, Museen und anderen öffentlichen Räumen verwendet, um den Fußverkehr zu überwachen und wertvolle Daten für die Analyse zu sammeln.

Um dieses Modul nutzen zu können, stellen Sie sicher, dass Ihre Geräte die Firmware Version v2.2.17 oder höher enthalten.

:::info
Dieses Modul kann nur mit ESP32 Geräten verwendet werden. Um das Besucherzählermodul zu betreiben, ist es zwingend erforderlich, sowohl WLAN als auch Bluetooth in Ihren Netzwerk- und Bluetooth-Einstellungen auszuschalten.
:::

## Besucherzählermodul Einstellung

### Aktiviert

Gibt an, ob das Modul aktiviert ist.

### Aktualisierungsintervall

Das Intervall in Sekunden, wie oft eine Nachricht an das Netz gesendet wird, wenn eine Statusänderung erkannt wurde.

#### Kommandozeile

:::info

Alle Konfigurationsoptionen des Besucherzählermoduls sind in der Python CLI Version v2.2.16 und höher verfügbar.

:::

Beispielbefehle sind folgende:

```shell title="Enable/Disable the Paxcounter Module"
meshtastic --set paxcounter.enabled true
meshtastic --set paxcounter.enabled false
```

```shell title="Set the Minimum Broadcast Interval to 900 seconds"
meshtastic --set paxcounter.paxcounter_update_interval 900
```

```shell title="Get the Paxcounter Module Configuration"
meshtastic --get paxcounter
```

</TabItem>

Module sind in der Firmware enthalten und erlauben es dem Benutzer, die Funktionalität des Netzes oder Gerätes zu erweitern.

|                      Name                                                         |                                                                                     Beschreibung                                                                                                                                                                                                 |
| :-------------------------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|      [Umgebungsbeleuchtung](/docs/configuration/module/ambient-lighting)          |                                                                   Helligkeit einer mit NCP5623 über I2C gesteuerten RGB LED anpassen.                                                                                                                                                            |
|             [Audio](/docs/configuration/module/audio)                             |                                                                        Codec2 Sprachkommunikation auf bestimmten Geräten aktivieren.                                                                                                                                                             |
|   [Vordefinierte Nachrichten](/docs/configuration/module/canned-message)          |                                   Legen Sie eine bestimmte Anzahl vordefinierter Nachrichten fest, die direkt vom Gerät die Mithilfe eines Eingabegerätes, wie z.B. eines Drehgebers, gesendet werden können.                                                                                    |
|        [Erkennungssensor](/docs/configuration/module/detection-sensor)            |                                                  Konfigurieren Sie einen GPIO Pin, dessen Status überwacht wird und in Abhängigkeit davon Textnachrichten versenden kann.                                                                                                                        |
|    [Externe Benachrichtigung](/docs/configuration/module/external-notification)   |                                                      Eingehende Nachrichten können Sie über Elemente, die Sie an das Gerät anschließen (LEDs, Summer, etc.), benachrichtigen.                                                                                                                    |
|              [MQTT](/docs/configuration/module/mqtt)                              |                                        Nachrichten an einen MQTT Server weiterleiten. Dies ermöglicht es Benutzern, vom lokalen Netz über das Internet mit Benutzern in einem anderen Netz zu kommunizieren.                                                                                     |
|       [Nachbarinformation](/docs/configuration/module/neighbor-info)              |                                                              Senden Sie Informationen über direkte (0-Sprung) Nachbarn an das Netz.                                                                                                                                                              |
|         [Besucherzähler](/docs/configuration/module/paxcounter)                   |                                                                  Zählt die Anzahl der BLE und WLAN Geräte, die an einem Knoten vorbeikommen.                                                                                                                                                     |
|        [Reichweitentest](/docs/configuration/module/range-test)                   | Senden Sie Nachrichten mit GPS-Standort in einem Intervall, um die Entfernung zu ermitteln, auf die Ihre Geräte kommunizieren können. Erfordert (jeweils) ein Gerät, das als Sender und Empfänger eingerichtet ist. Der Empfänger protokolliert alle eingehenden Nachrichten in einer CSV Datei. |
|      [Abgesetzte Hardware](/docs/configuration/module/remote-hardware)            |                                                                Legen Sie einen GPIO Pinstatus über dem Netz fest und lesen Sie ihn aus.                                                                                                                                                          |
|          [Serielles Modul](/docs/configuration/module/serial)                     |                                                               Senden Sie Zeichenketten über einen seriellen Anschluss als Nachrichten an das Netz.                                                                                                                                               |
| [Speichern & Weiterleiten](/docs/configuration/module/store-and-forward-module)   |                                                        Speichert Nachrichten auf einem Gerät zur Weiterleitung, nachdem Knoten wieder mit dem Netz verbunden sind.                                                                                                                               |
|           [Telemetrie](/docs/configuration/module/telemetry)                      |                                                        Schließen Sie Sensoren an das Gerät an und senden Sie Messwerte in einem regelmäßigen Intervall an das Netz.                                                                                                                              |
|        [Routenverfolgung](/docs/configuration/module/traceroute)                  |                                                      Verfolgen Sie, welche Knoten verwendet werden, um eine Nachricht an ein bestimmtes Ziel weiter zuleiten.                                                                                                                                    |

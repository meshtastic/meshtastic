---

Es gibt mehrere Konfigurationsabschnitte in der Meshtastic Firmware, diese sind so aufgeteilt, dass sie als kleine administrative Nachrichten über das Netzwerk gesendet werden können.

|         Name                   |                                                             Beschreibung                                                                                                                                                   |
| :----------------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|    [Bluetooth](./bluetooth)    |                                        Die Bluetooth-Konfigurationsoptionen sind: Aktiviert, Kopplungsmodus und feste Pin.                                                                                                 |
|      [Kanäle](./channels)      |                                         Die Konfigurationsoptionen der Kanäle sind: Index, Rolle und Einstellungen.                                                                                                        |
|      [Gerät](./device)         |                                Die Gerätekonfigurationsoptionen sind: Geräterolle, serielle Ausgabe, Debug-Protokoll und Werkseinstellung.                                                                                 |
|     [Anzeige](./display)       |                                        Die Anzeigeeinstellungen sind: Bildschirmdauer, Karussell-Intervall, Kompassorientierung und GPS Format.                                                                            |
|       [LoRa](./lora)           |   Die LoRa Konfigurationsoptionen sind: Region, Modem Voreinstellung, maximale Sprungweite, Sendeleistung, Bandbreite, Spreizfaktor, Kodierungsrate, Frequenzversatz, Sendebetrieb deaktiviert und Knoten ignorieren.      |
|     [Netzwerk](./network)      |                                Die Netzwerkkonfigurationsoptionen sind: WiFi aktiviert, WiFi SSID, WiFi Schlüssel, WiFi Mode und NTP-Server.                                                                               |
|     [Standort](./position)     |            Die Konfigurationsoptionen für den Standort sind: GPS aktiviert, GPS Aktualisierungsintervall, GPS Suchzeit (veraltet), fester Standort, Smart Position, Sendeintervall und PositionFlags.                      |
| [Stromversorgung](./power)     |      Die Optionen für die Stromversorgungskonfiguration sind: Ladestrom, Energieeinsparung, Herunterfahren nach Stromverlust, ADC Multiplikationsfaktor, Bluetooth Intervall, Schlafmodus und minimale Aufwachperiode.     |
|    [Sicherheit](./security)    | Die Konfigurationsoptionen zur Sicherheit sind: Öffentlicher Schlüssel, privater Schlüssel, administrative Schlüssel, Serielle Konsole, Debug-Protokoll und administrativer Kanal (veraltet).                              |
|     [Benutzer](./user)         |                                Die Konfigurationsoptionen des Benutzers sind: Langer Name, kurzer Name und die Amateurfunklizenz.                                                                                          |

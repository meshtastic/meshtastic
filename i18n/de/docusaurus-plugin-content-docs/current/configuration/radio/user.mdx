Die Konfigurationsoptionen des Benutzers sind: langer Name, kurzer Name, lizenzierter Funkamateur und ist nicht erreichbar. Die Benutzereinstellung verwendet eine administrative Nachricht, die einen `User` Protocol Buffer sendet.

## Benutzereinstellungen

### Langer Name

Ein personalisierter Name für Ihr Gerät

Automatisch erzeugt.

Wenn Sie ein lizenzierter Funkamateur sind und `Amateurfunklizenz` aktiviert haben, sollte dies auf Ihr Rufzeichen gesetzt werden.

### Kurzname

Eine personalisierte Kurzkennung für Ihr Gerät.

Automatisch erzeugt.

### Amateurfunklizenz

Wenn Sie ein lizenzierter Amateurfunker sind und die [Privilegien und Einschränkungen](/docs/faq/#amateur-radio-ham) in Betracht gezogen haben, Meshtastic mit ihrer Lizenz zu benutzen, dann aktivieren Sie diese Option.

Standardmäßig deaktiviert

Wenn Sie `Amateurfunklizenz` aktivieren, sollten Sie auch die folgenden verwandten Einstellungen überprüfen:

- **Benutzer:** `Langer Name` (sollte ihr Rufzeichen sein)
- **Kanal:** `PSK Schlüssel` (sollte leer sein, um die Verschlüsselung zu entfernen)

### Nicht erreichbar

Gibt an, ob der Knoten direkte Nachrichten empfangen kann oder nicht.

Standardmäßig deaktiviert, außer Rollen, die normalerweise nicht auf Direktnachrichten überwacht werden: ROUTER, ROUTER_LATE, REPEATER, SENSOR, LOST_AND_FOUND, TRACKER, TAK_TRACKER

#### Kommandozeile

:::info

Alle Benutzereinstellungen sind über die Python Kommandozeile verfügbar. Beispieleinstellungen sind folgende:

:::

Bitte lesen Sie sich die Anweisungen um die [Amateurfunklizenz zu aktivieren](/docs/software/python/cli/usage#ham-radio-support)

:::tip

Weil das Gerät nach jedem Befehl über die Kommandozeile neu startet, wird empfohlen, dass mehrere Werte in einem Konfigurationsblock zusammengefasst und übertragen werden.

```shell title="Example:"
meshtastic --set-owner 'Ihr Knotenname' --set-owner-short 'NODE'
```

:::

```shell title="Set the LongName Value"
meshtastic --set-owner 'Ihr Knotenname'
```

```shell title="Set the ShortName Value"
meshtastic --set-owner-short 'NODE'
```

```shell title="Enable Ham Mode, set name to license, disable encryption"
meshtastic  --set-ham 'Ihr Rufzeichen'
```

Unten finden Sie eine Liste von Gruppen, die sich aktiv für die Organisation von Meshtastic Netzwerken in ihren Regionen einsetzen. Kontaktieren Sie diese um Hilfe beim Einstieg zu erhalten oder wenn Sie interessiert sind am Netzwerk mitzuwirken. Wenn Sie ein Gruppenorganisator mit Online Präsenz sind und in diese Liste aufgenommen werden möchten, dann [bearbeiten Sie diese Seite direkt](#edit-this-page) oder kontaktieren Sie uns über [Discord](https://discord.com/invite/ktMAKGBnBs), um Ihre Gruppe hinzuzufügen.

:::info Voraussetzungen zur Einreichung

- Gruppen können das Meshtastic Logo verwenden, wenn es sich in einer sekundären Position im primären Auftritt der Gruppe befindet. Im Zweifelsfall sollte stattdessen das [M-Powered Logo](/docs/legal/licensing-and-trademark/#community-project-logos-m-powered--m-pwrd) oder ein benutzerdefiniertes Design verwendet werden.
- Gruppen dürfen "Meshtastic [Ihre Stadt hier]" nicht als Link, Gruppenname oder Seitentitel verwenden, da dies auf ein offizielles Sponsoring hindeuten kann.
- Server-Einladungslinks dürfen niemals ablaufen.
  :::

Wenn du einen Discord Server für deine lokale Community betreibst, folge unserem Ankündigungskanal, um wichtige Updates direkt auf deinem Server zu erhalten. Lernen Sie, wie sie unserem [Discord Kanal folgen](https://support.discord.com/hc/en-us/articles/360028384531-Channel-Following-FAQ#:~:text=Adding%20It%20To%20Your%20Server).

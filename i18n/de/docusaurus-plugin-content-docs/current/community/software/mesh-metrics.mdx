:::note

Dies ist ein Gemeinschaftsprojekt von @cordelster.
Entwicklung kann auf [GitHub](https://github.com/cordelster/mesh-metrics) verfolgt werden.
Unterstützung sollte bei den jeweiligen Autoren erfragt werden.

:::

Mesh-metrics ist ein Skript, welches das Meshtastic Kommandozeileninterface und ein über serielles USB angeschlossenes Gerät verwendet, um Metrikanfragen aus einer Liste von Geräten anzufordern und die Ergebnisse in einem Format auszugeben, das von node_exporter gelesen werden kann, um eine Prometheus/Victoria Metrics Datenbank zu füllen. Zusammen mit dem Skript sind eine Grafana Oberfläche und einige Alarmierungsbeispiele enthalten. Die Mesh-Metriken wurden erstellt und wird von Cordel gepflegt. Weitere Informationen zur Verwendung und Einrichtung finden Sie auf der [Mesh-Metriken Github Seite](https://github.com/cordelster/mesh-metrics)

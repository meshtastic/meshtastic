## Erstellt von tavdog/Tavis Gustafson

### Hack für ein billiges Gehäuse aus einer Harbor Breeze Solarlampe für RAK

![Solarlampen Hack fertig](/img/enclosures/hbmh_complete.webp)

Die Harbor Breeze LED Solarlampe kann für etwa $15 gekauft werden und enthält ein ausreichendes Solarpanel, Lithium-Ionen Akku und Laderegler in einem wasserdichten Gehäuse. Die RAK Basisplatinen passen perfekt in das Solarfach. Unten sind die Schritte für die Fertigstellung dieses Hacks für einen komplett wetterfesten Meshtastic Knoten.

### Erforderliche Fertigkeiten

Es muss gelötet und ein Loch in das Kunststoffgehäuse bohren werden.

### Benötigte Bauteile

Für eine externe Antenne ist ein IPEX zu SMA Adapterkabel erforderlich. Ein JST-PHR-2-Anschluss wird für eine Batterieanbindung zum Anstecken benötigt. Gelötete Verbindungen funktionieren aber auch. Dichtmittel (Silikon oder aus dem Schiffbau)

### Schritte

Schritt 1 Entfernen Sie die 4 kleine Schrauben, um das Solarfach zu öffnen.

Schritt 2 Löten Sie Ihren JST-PHR-2-Stecker oder Stromkabel an die + und - Anschlüsse der Batterie. Wenn Sie nicht über den richtigen Stecker verfügen, können Sie auch direkt an den Pins auf der Rückseite des JST-Steckers löten. Stellen Sie sicher, dass die Polarität korrekt ist.
![Löten der Batterieanschlüsse](/img/enclosures/harbor_breeze_solar_battery_solder_points.webp)

Schritt 3 Bohre Sie ein Loch für die SMA Buchse. Eine perfekt vertikale Antenne durch eine 90 Grad Biegung erlaubt keine sehr flache Solarplatte, treffen Sie eine gute Wahl oder verwenden Sie eine externe Antenne. Sie können diesen Schritt überspringen, wenn Sie eine interne Antenne verwenden möchten, die passt.
![Loch für SMA Buchse bohren](/img/enclosures/harbor_breeze_solar_sma_hole1.webp)![Loch für SMA Buchse bohren](/img/enclosures/harbor_breeze_solar_sma_hole2.webp)

Schritt 4 Bauen Sie die SMA Buchse ein. Dichten Sie diese mit Silikon oder anderer Dichtmasse ab.
![SMA Buchse einbauen](/img/enclosures/harbor_breeze_solar_sma_installed.webp)![Abdichtung](/img/enclosures/harbor_breeze_solar_sma_seal.webp)

Schritt 5 Bauen Sie das RAK Modul im freien Platz ein. Wenn Sie möchten, können Sie die Platine leicht an der Stelle verkleben.
![Einbau der RAK Platine](/img/enclosures/harbor_breeze_solar_board_placement.webp)

Schritt 6 Schließen Sie die Antenne sowie den Akku an und verschließen Sie die Rückseite.

Das Lampengehäuse kann bei Bedarf entfernt werden.
![Externe Antenne über SMA Adapter](/img/enclosures/harbor_breeze_solar_antenna_pigtail.webp)

Das Lampengehäuse ist nicht so wasserdicht wie das Solargehäuse also seien Sie vorsichtig, wenn Sie Elektronik dort einbauen.
An einem Stab befestigen und hoch aufstellen.
![Externe Antenne über SMA Adapter](/img/enclosures/harbor_breeze_solar_stick_mount.webp)

Das Licht wird sich nachts für einen sehr gut getarnten Knoten einschalten. Um das Licht auszuschalten, gibt es einen Knopf auf der Rückseite.

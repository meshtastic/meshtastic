## Offlinekarten

### Apple Karten

Standardmäßig unterstützt die Meshtastic App für iOS die Verwendung von Apple Offline Karten.

Um Offline Karten herunterzuladen, folgen Sie den offiziellen Anweisungen von Apple für [iPhone](https://support.apple.com/en-gb/105084) oder [iPad](https://support.apple.com/en-gb/guide/ipad/ipadc6e7e4d7/ipados. Videoanleitungen siehe auch [available](https://youtube.com/watch?v=L260qixQrYs).

## Verknüpfungen und Siri

Die Meshtastic-App enthält Integrationen mit Siri und Shortcuts, Sie können Aktionen wie das Senden von Wegpunkten oder das Neustarten eines Knoten durch automatisierte Workflows durchführen.

### Verknüpfungen

Sie können eine Verknüpfung erstellen, indem Sie die Shortcuts App öffnen, eine neue Verknüpfung erstellen und nach Aktionen suchen, die in der Meshtastic App aufgelistet werden

Alle Verknüpfungen:

- Kontakt hinzufügen
  - Verwendet eine Meshtastic Kontakt-URL und speichert sie in der Knotendatenbank
- Eine Direktnachricht senden
  - Sende eine Nachricht an einen bestimmten Meshtastic Knoten
- Wegpunkt senden
- Gruppennachricht senden
  - Sende eine Nachricht an einen bestimmten Meshtastic Kanal
- Knotenposition ermitteln
  - Letzte Position eines Knotens holen
- Herunterfahren
  - Herunterfahren an verbundenen Knoten senden
- Neustarten
  - Verbundenen Knoten neustarten
- Werkseinstellungen
  - Verbundenen Knoten auf Werkseinstellungen zurücksetzen
- Kanaleinstellungen speichern
  - Verwendet eine Meshtastic Kanal URL und speichert die Kanaleinstellungen
- Knoten trennen
  - Den aktuell verbundenen Knoten trennen

### Siri

Sie können die folgenden Siri Aktionen ausführen, indem Sie den Satz nach der Aktivierung von Siri sprechen.

---

#### Knoten herunterfahren

Phrasen:

- Meshtastic Knoten herunterfahren
- Meinen Meshtastic Knoten herunterfahren
- Meshtastic Knoten ausschalten
- Meshtastic Knoten herunterfahren
- Meshtastic Knoten deaktivieren

---

#### Knoten neu starten

Phrasen:

- Meshtastic Knoten neu starten
- Meinen Meshtastic Knoten neu starten
- Meshtastic Knoten neu starten
- Meshtastic Knoten neu starten

---

#### Gruppennachricht

Phrasen:

- Nachricht an einen Meshtastic Kanal
- Sende eine Meshtastic Gruppennachricht

---

#### Knoten trennen

Phrasen:

- Meshtastic Knoten trennen
- Meinen Meshtastic Knoten trennen
- Verbindung von Meshtastic trennen
- Meshtastic trennen

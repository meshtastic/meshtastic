## Wie man beitragen kann

Übersetzungen in die Meshtastic Apple App beitragen hilft, das Projekt einem größeren Publikum zugänglich zu machen. Die Meshtastic Apple App verwendet [Kataloge der Zeichenfolge](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) in Xcode, um Übersetzungen zu verwalten. 

Wenn Sie die Übersetzungen für eine Sprache aktualisieren oder eine neue Sprache hinzufügen möchten, folgen Sie diesen Schritten.

1. Beginnen Sie mit dem Forking des [Meshtastic-Apple Repository](https://github.com/meshtastic/Meshtastic-Apple/tree/main) in Ihrem GitHub Konto.
2. Klonen Sie das Projekt und öffnen Sie die Datei `Meshtastic.xcworkspace`
3. Wählen Sie die Datei `Localizable.xcstrings` im Projektnavigator.
4. Folgen Sie den Schritten, die [hier](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) beschrieben sind, um Übersetzungen hinzuzufügen oder zu aktualisieren.
5. Erstellen Sie mit Ihren Änderungen eine PR im Projekt.

Ihr Beitrag wird überprüft und nach Genehmigung wird Ihre Übersetzung in die nächste Version der Meshtastic Apple App aufgenommen. Vielen Dank, dass Sie dabei geholfen haben, die Reichweite von Meshtastic zu vergrößern!

Das Meshtastic Ökosystem ist hochgradig erweiterbar und ermöglicht eine einfache Integration in eine Reihe bestehender Softwareprodukte und Projekte.

Aktuelle Meshtastische Integrationen:

- [CalTopo / SARTopo](/docs/software/integrations/integrations-caltopo) - Verfolgt Meshtastic Knoten in CalTopo und SARTopo.

- [ATAK Plugin](/docs/software/integrations/integrations-atak-plugin) - Offizielles Meshtastic ATAK Plugin für das Senden von CoT an IMeshService in der Meshtastic Android App.

- Netzwerke über das Internet überbrücken und Meshtastic Protokolle mit populären Technologien wie Home Assistant, Node Red und Adafruit IO integrieren.

Unterstützung für die integrierten Produkte sollte bei den jeweiligen Autoren oder Anbietern gesucht werden.

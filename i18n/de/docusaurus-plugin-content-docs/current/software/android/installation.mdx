## Installationsmethode

Unsere Android-Anwendung ist im Repository F-Droid und Google Play Store verfügbar. Dadurch können Sie sich mit Ihrem Meshtastic Gerät über Bluetooth, Wi-Fi (falls im selben Netzwerk) oder USB On-The-Go (OTG) verbinden.

Die minimale Android-Version ist 8.0 (Oreo 2017).

### Mit F-Droid installieren

<p float="center">
  <a href="https://f-droid.org/packages/com.geeksville.mesh/">
    <img
      alt="Get it on F-Droid"
      src="https://fdroid.gitlab.io/artwork/badge/get-it-on.png"
      width="49%"
    />
  </a>
</p>

1. Laden und installieren Sie die F-Droid App von [f-droid.org](https://f-droid.org/)
2. Navigiere zur Seite `Kategorien` und aktualisieren sie die Packetliste (scrollen nach unten).
3. Suchen und installieren Sie die `Meshtastic` App.

Die App kann auch durch das Herunterladen der .APK von der [Github Releases](https://github.com/meshtastic/Meshtastic-Android/releases/latest) Seite installiert werden.

Wenn Sie den direkten .APK Download verwenden, müssen Sie möglicherweise Ihrem Browser Berechtigungen geben, um die Paketinstallation ausführen zu können. Wenn Sie den Quellcode einsehen oder zur Entwicklung der App beitragen möchten, dann besuchen Sie bitte die [GitHub-Seite] (https://github.com/meshtastic/Meshtastic-Android).

### Mit Obtainium installieren

1. Downloaden und installieren Sie die Obtainium App von [Github](https://github.com/ImranR98/Obtainium).
2. Öffnen Sie die Obtainium App und navigieren Sie zu `+ App hinzufügen`.
3. Sie können ganz einfach nach dem Android Repository im suchen, indem Sie `Meshtastic-Android` eingeben und das `Meshtastic/Meshtastic-Android` Repository auswählen. Sie können auch manuell die Meshtastic Android Github Releases Adresse wie folgt eingeben: `https://github.com/meshtastic/Meshtastic-Android/releases`.
4. Unter `Zusätzliche Optionen für Github` wechseln Sie wie gewünscht `Include prereleases`\* oder `Fallback zu älteren Releases` und drücken `Add`.
5. Wenn Sie eine Anwendung zum ersten Mal hinzufügen, werden Sie aufgefordert, unbekannte Apps zu installieren. Sie müssen `Erlaube von dieser Quelle` aktivieren und auf "Zurück" drücken. Obtainium wird die Android .APK von der Github Release Seite herunterladen.
6. Drücken Sie \`Installieren'. Der Android Installer wird fragen:"Möchten Sie diese App installieren?" wählen Sie dann "Installieren".
7. Drücken Sie `Öffnen`.

Mit Obtainium können Sie Open-Source-Apps direkt von ihren Veröffentlichungsseiten aus installieren, aktualisieren und erhalten Benachrichtigungen, wenn neue Veröffentlichungen verfügbar sind.

\*Alpha Versionen enthalten die neuesten Änderungen, die mit zusätzlichen Fehlern einhergehen können. Es wird empfohlen, dem Meshtastic Alpha Testers Kanal auf [Discord] zu folgen (https://discord.com/invite/ktMAKGBnBs), wenn Sie diese Versionen verwenden.

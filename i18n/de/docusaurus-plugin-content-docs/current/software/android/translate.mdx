## Wie man beitragen kann

Übersetzungen in die Meshtastic Android-App beitragen hilft, das Projekt einem größeren Publikum zugänglich zu machen. Folgen Sie diesen Schritten, um Ihre Übersetzungen über Crowdin hinzuzufügen:

1. **Zugriff auf Crowdin**: Besuchen Sie die Crowdin Seite des Meshtastic Projekts unter [https://crowdin.meshtastic.org](https://crowdin.meshtastic.org).
2. **Erstellen Sie ein Konto**: Klicken Sie auf 'Anmelden' in der oberen rechten Ecke der Seite und folgen Sie den Anfragen, um ein Crowdin-Konto zu erstellen.
3. **Navigiere zum Projekt**: Sobald du eingeloggt bist, suche und wähle das Projekt 'Android Anwendung' aus dem Crowdin-Dashboard.
4. **Wählen Sie eine Sprache**: Finden Sie die Sprache, für die Sie Übersetzungen beisteuern möchten, und klicken Sie auf 'Zum Editor', um mit der Übersetzung zu beginnen.
5. **Starte Übersetzung**: Im Editor sehen Sie eine Liste von Zeichenketten auf der linken Seite. Klicken Sie auf eine Zeichenkette, um sie auszuwählen und geben Sie dann Ihre Übersetzung in das Editorfeld ein. Wenn Sie mit einer Zeichenkette fertig sind, klicken Sie auf 'Speichern', um Ihre Übersetzung zu speichern. Wiederholen Sie diesen Prozess für jede Zeichenkette, die Sie übersetzen möchten.

Ihr Beitrag wird überprüft und nach Genehmigung wird Ihre Übersetzung in die nächste Version der Meshtastic Android App aufgenommen. Vielen Dank, dass Sie dabei geholfen haben, die Reichweite von Meshtastic zu vergrößern!

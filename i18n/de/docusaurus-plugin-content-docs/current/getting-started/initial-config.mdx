## Regionale Einstellungen

Um mit dem Netz zu kommunizieren, müssen Sie eine Region auswählen. Diese Einstellung bestimmt, welchen Frequenzbereich Ihr Gerät nutzt und sollte entsprechend Ihrem Standort eingestellt werden.

<TabItem value="android">

#### Android

1. Befolgen Sie die Anweisungen zur [Installation](/docs/software/android/installation) und [Nutzung](/docs/software/android/usage/) für [Meshtastic Android](/docs/category/android-app).
2. Öffnen Sie die App und stellen Sie von Ihrem Telefon aus über USB, serielle Schnittstelle oder Bluetooth eine Verbindung zum Gerät her.
3. Klicken Sie nach der Kopplung neben dem Gerätenamen auf „UNSET“.
4. Wählen Sie aus der Liste die Region entsprechend Ihrem regionalen Standort aus.

</TabItem>
<TabItem value="apple">

#### Apple

:::info
Die Einstellungen von Region, Modemvoreinstellung und Sprungweite ist unter iOS, iPadOS und macOS unter Einstellungen > Funkkonfiguration > LoRa verfügbar.
:::

</TabItem>
<TabItem value="cli">

#### Kommandozeile

1. Installieren Sie [Meshtastic PythonCLI](/docs/software/python/cli/installation)
   ```sh
   pip3 install --upgrade pytap2
   pip3 install --upgrade meshtastic
   ```
2. Führen Sie den folgenden Befehl aus und ersetzen Sie `<REGION-CODE>` durch den oben aufgeführten Regionalcode entsprechend Ihrem regionalen Standort.
   ```sh
   meshtastic --set lora.region <REGION-CODE>
   ```

</TabItem>
<TabItem value="web">

#### Web

1. Öffnen Sie die Meshtastic Weboberfläche: [client.meshtastic.org](https://client.meshtastic.org)
2. Navigieren Sie zum **LoRa** Menü.
3. Legen Sie unter **regionale Einstellungen** Ihre **Region** entsprechend Ihrem regionalen Standort fest.
4. Klicken Sie **Speichern**.

</TabItem>
</Tabs>

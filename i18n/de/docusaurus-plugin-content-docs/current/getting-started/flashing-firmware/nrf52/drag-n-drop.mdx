## Firmware programmieren

:::info
Sie können nun den [Meshtastic Web Flasher](https://flasher.meshtastic.org) verwenden, um Firmware auf Ihre nRF52- oder RP2040-basierten Geräte herunterzuladen und zu kopieren. Alternativ folgen Sie den Anweisungen unten, um Firmware herunterzuladen und zu installieren.
:::

Wenn Sie Ihr Gerät auf [Werkseinstellungen zurückgestetzt](/docs/getting-started/flashing-firmware/nrf52/nrf52-erase/) haben, dann Ihr Gerät automatisch im Bootloader Modus automatisch, und Sie können zu "Kopieren Sie die entsprechende Firmware" für Ihr Gerät nach unten springen:

### nRF52

1. Laden und entpacken Sie die neueste Firmware von [Meshtastic Downloads](https://meshtastic.org/downloads).
2. Verbinden Sie Ihr Gerät mit einem USB-Datenkabel.
3. Doppelklicken Sie auf den Reset Button auf Ihrem Gerät (um den Bootloader Modus zu aktivieren).
4. Beachten Sie, dass ein neues Laufwerk auf Ihrem Computer (Windows, Mac, Linux oder Android) eingebunden wird.
5. Öffnen Sie dieses Laufwerk und Sie sollten drei Dateien sehen: `CURRENT.UF2`, `INDEX.HTML und `INFO_UF2.TXT\`.
6. Kopieren Sie die entsprechende Firmware-Datei (`firmware-DEVICE_NAME-X.X.X-xxxxx.uf2`) aus der Veröffentlichung auf dieses Laufwerk.
7. Sobald die Datei auf das Laufwerk kopiert wurde, wird das Gerät neu starten und die Meshtastic Firmware installieren.

### RP2040

1. Laden und entpacken Sie die neueste Firmware von [Meshtastic Downloads](https://meshtastic.org/downloads).
2. Drücken Sie die BOOTSEL Taste und verbinden Sie das Gerät über ein USB-Datenkabel.
3. Beachten Sie, dass ein neues Laufwerk auf Ihrem Computer (Windows, Mac, Linux oder Android) eingebunden wird.
4. Öffnen Sie dieses Laufwerk und Sie sollten zwei Dateien sehen: `INDEX.HTM` und `INFO_UF2.TXT`.
5. Kopieren Sie die entsprechende Firmware-Datei (`firmware-DEVICE_NAME-X.X.X-xxxxx.uf2`) aus der Veröffentlichung auf dieses Laufwerk.
6. Sobald die Datei auf das Laufwerk kopiert wurde, wird das Gerät neu starten und die Meshtastic Firmware installieren.

### Potenzielle Problemfälle bei der Firmware Aktualisierung

:::info

Bevor Sie aktualisieren, prüfen Sie bitte, dass Sie ein [RAK4631](https://docs.rakwireless.com/product-categories/wisblock/rak4631/overview) und kein [RAK4631-R](https://docs.rakwireless.com/product-categories/wisblock/rak4631-r/overview) haben. Machen Sie sich keine Sorgen wenn dies nicht der Fall ist. Die Hardware ist identisch, es muss jedoch den Bootloader geändert werden. Anweisungen was zu tun ist, finden Sie [hier](/docs/getting-started/flashing-firmware/nrf52/convert-rak4631r).

:::

:::info

Vorherige Versionen der Meshtastic Firmware können veraltete Daten speichern, was dazu führt, dass Geräte beim Start in einer Absturzschleife stecken bleiben. Wenn Sie Probleme beim Aktualisieren Ihres nRF52 Gerätes von einer vorherigen Version von Meshtastic haben, dann müssen Sie eventuell eine vollständige Löschung auf Werkseinstellungen des internen Speichers durchführen.

Folgen Sie dazu der Anleitung [Werkseinstellungen für nRF52](/docs/getting-started/flashing-firmware/nrf52/nrf52-erase) Geräte, bevor Sie mit [Firmware Aktualisierung](#flash-firmware) fortfahren.

:::

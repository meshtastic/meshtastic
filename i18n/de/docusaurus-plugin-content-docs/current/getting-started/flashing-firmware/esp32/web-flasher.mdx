## Gerät programmieren

1. Schließen Sie Ihr Gerät an
2. Besuchen Sie [flasher.meshtastic.org](https://flasher.meshtastic.org) _\*erfordert Chrome oder Edge Browser_
3. Folgen Sie der Anleitung

### Gerät verbinden und konfigurieren

### Serielle Treiberinstallation testen

Sie können überprüfen, ob Sie ein geeignetes Datenkabel haben (statt ein Lade-Typenkabell) und dass die passenden Treiber für Ihr System installiert werden, indem Sie folgende Tests durchführen:

#### Linux

1. Verbinden Sie Ihr Meshtastic Gerät mit Ihrem USB-Anschluss

2. Öffnen Sie ein **Terminal** und geben Sie folgenden Befehl ein:

   ```shell
   lSusb
   ```

3. Sie sollten etwas sehen wie:

   ```shell
   ID xxxx:xxxx Silicon Labs CP210x UART Bridge
   # oder
   ID xxxx:xxxx QinHeng Electronics USB Single Serial
   # oder
   ID xxxx:xxxx Adafruit WisCore RAK4631 Board
   ```

</TabItem>
<TabItem value="macos">

#### macOS

1. Navigieren Sie zu `Apple Menü > Über diesen Mac > Mehr Info > Systembericht... > Hardware > USB`.
2. Sie sollten einen der folgenden Einträge oder Ähnliches sehen:

- `CP210X USB to UART Bridge Controller`
- `CH9102 USB to UART Bridge Controller`
- `WisCore RAK4631 Board`
- `USB Single Serial`

</TabItem>
<TabItem value="windows">

#### Windows

1. Navigieren Sie zu `Gerätemanager > Anschlüsse (COM & LPT)`
2. Sie sollten einen der folgenden Einträge oder Ähnliches sehen:

- `Silicon Labs CP210X USB to UART Bridge (COM5)`
- `Silicon Labs CH9102 USB to UART Bridge (COM5)`
- `USB-Enhanced-SERIAL CH9102 (COM5)`
- `USB Serial Device (COM5)`

</TabItem>
</Tabs>

:::info

Wenn Sie Ihr Gerät nicht sehen können:

- Stellen Sie sicher, dass Ihr Kabel nicht nur zum Laden, sondern auch für [Datenübertragung](/docs/getting-started#verify-data-cable) geiegnet ist.
- Es ist möglich, dass Sie [den seriellen USB-Treiber neu installieren](/docs/getting-started/serial-drivers) müssen.

:::

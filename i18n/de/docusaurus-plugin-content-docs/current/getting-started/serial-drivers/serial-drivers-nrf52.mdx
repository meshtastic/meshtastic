## Installieren Sie den nRF52/RP2040 USB zu seriellen Treiber

:::caution

nRF52/RP2040 Geräte benötigen normalerweise keine seriellen Treiber. Sie verwenden den UF2 Bootloader, der die Geräte als USB Laufwerke erscheinen lässt. Laden Sie die USB-Gerätetreiber _nicht herunter_, es sei denn, sie müssen die UF2 Unterstützung installieren.

:::

##### Entfernen Sie den CH34x USB Treiber (macOS)

Wenn Sie die macOS WCH-IC CH340/CH341
("CH341SER_MAC") Treiber über die Datei CH34x_Install_V1.5.pkg bereits heruntergeladen haben, müssen Sie die Kernel Erweiterung deinstallieren:

1. Gerät trennen
2. Öffne eine Kommandozeile und führe folgenden Befehl aus:
3. `sudo rm -rf /Library/Extensions/usbserial.kext`
4. Neustarten

##### Installieren Sie den CH34x-Treiber

- [CH34x Treiber- macOS Download](https://github.com/WCHSoftGroup/ch34xser_macos)

Sie können auch [Ihre serielle Treiberinstallation](/docs/getting-started/serial-drivers/test-serial-driver-installation) bei Bedarf mit diesem Schritt testen.

:::

### Firmware programmieren

Nach der Installation der seriellen Treiber können Sie nun die Meshtastic Firmware auf Ihrem Gerät programmieren. Um fortzufahren, wählen Sie den entsprechenden Gerätetyp für Ihr Gerät.

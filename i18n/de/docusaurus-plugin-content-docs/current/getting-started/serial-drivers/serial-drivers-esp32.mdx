## ESP32 USB zu serielle Treiber installieren

Möglicherweise müssen Sie einen Treiber von Silicon Labs für den [CP210X USB zu UART Adapter](https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers) installieren

Einige neuere Platinen benötigen möglicherweise den CH9102 (CH340/CH341) Treiber.

:::important

Nach der Installation des Treibers stellen Sie sicher, dass Sie Ihren Computer neu starten, um den Installationsprozess abzuschließen.

Sie können auch [Ihre serielle Treiberinstallation](/docs/getting-started/serial-drivers/test-serial-driver-installation) bei Bedarf mit diesem Schritt testen.

:::

### Firmware programmieren

Nach der Installation der seriellen Treiber können Sie nun die Meshtastic Firmware auf Ihrem Gerät programmieren. Um fortzufahren, wählen Sie den passenden Gerätetyp für Ihr Gerät.

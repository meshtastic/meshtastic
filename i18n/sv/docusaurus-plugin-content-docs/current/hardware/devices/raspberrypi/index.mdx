Fast versatile boards using the RP2040/RP2350.

| :--------------------------- | :------------ | :----- | :--------------: | :-----------: | :-: |

[^1]: 2.4GHz b/g/n WiFi support is available on the Raspberry Pi Pico **W** variant.

## Ställ in regionala inställningar

För att börja kommunicera över meshen, måste du ställa in din region. Den här inställningen kontrollerar vilket frekvensområde din enhet använder och ska ställas in efter din region.

<TabItem value="android">

#### Android

1. Följ instruktionerna för [installation](/docs/software/android/installation) och [användning](/docs/software/android/usage/) för [Meshtastic Android](/docs/category/android-app).
2. Öppna appen, anslut till enheten från telefonen via USB (seriell kommunikation) eller Bluetooth.
3. När du har parkopplat enheten, klicka på "UNSET" bredvid enhetsnamnet.
4. Välj region från listan efter den plats du kommer använda enheten på.

</TabItem>
<TabItem value="apple">

#### Apple

:::info
Konfiguration av region, modeminställningar och hopp-gräns finns tillgänglig på iOS, iPadOS och macOS under Inställningar > Radiokonfiguration > LoRa.
:::

</TabItem>
<TabItem value="cli">

#### CLI

1. Installera [Meshtastic Python CLI](/docs/software/python/cli/installation)
   ```sh
   pip3 install --upgrade pytap2
   pip3 install --upgrade meshtastic
   ```
2. Kör följande kommando, ersätt `<REGIONSKOD>` med regionkoden som anges ovan enligt din region.
   ```sh
   meshtastic --set lora.region <REGIONSKOD>
   ```

</TabItem>
<TabItem value="web">

#### Webb

1. Öppna webbgränssnittet: [client.meshtastic.org](https://client.meshtastic.org)
2. Navigera till **LoRa**-menyn.
3. Under **Regionala inställningar**, ange din **Region**.
4. Klicka på **Spara**.

</TabItem>
</Tabs>

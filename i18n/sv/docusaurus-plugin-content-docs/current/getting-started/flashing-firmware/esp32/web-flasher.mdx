## Installera programvara

1. Koppla in enheten
2. Besök [flasher.meshtastic.org](https://flasher.meshtastic.org) _\*kräver Chrome eller Edge webbläsare_
3. Följ anvisningarna

### Anslut och konfigurera enheten

Efter att ha installerat Meshtastic på din enhet, kan du nu gå vidare till de första inställningarna.

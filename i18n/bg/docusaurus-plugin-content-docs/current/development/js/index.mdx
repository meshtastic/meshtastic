## Въведение

Meshtastic.js е библиотека на JavaScript, която предоставя интерфейс към устройствата Meshtastic. Може да се използва за изграждане на приложения за взаимодействие с Meshtastic мрежа. Понастоящем се поддържат HTTP(S) и Bluetooth връзки.

Ако желаете да видите кода или да допринесете за развитието на библиотеката, моля, посетете кода на JavaScript на [страницата в GitHub](https://github.com/meshtastic/web).

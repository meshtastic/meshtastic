Във фърмуера на Meshtastic има няколко конфигурационни секции, които са разделени, така че могат да се изпращат като малки администраторски съобщения през mesh-а.

|       Име                 |                                                       Описание                                                                                                                                             |
| :-----------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|  [Bluetooth](./bluetooth) |                              Опциите за конфигуриране на Bluetooth са: Активиран, Режим на сдвояване и Фиксиран ПИН.                                                                                       |
|   [Канали](./channels)    |                                     Опциите за конфигуриране на каналите са: Индекс, Роля и Настройки.                                                                                                     |
| [Устройство](./device)    |                   Опциите за конфигуриране на устройството са: Роля на устройството, Сериен изход, Журнал на грешки и Фабрично нулиране.                                                                   |
|   [Дисплей](./display)    |                Опциите за конфигуриране на дисплея са: Продължителност на включен екран, Интервал на автоматична смяна, Винаги сочи север и GPS формат.                                                    |
|    [LoRa](./lora)         |           The LoRa config options are: Region, Modem Preset, Max Hops, Transmit Power, Bandwidth, Spread Factor, Coding Rate, Frequency Offset, Transmit Disabled and Ignore Incoming Array.               |
|    [Мрежа](./network)     |                           Опциите за конфигуриране на мрежата са: Активиран WiFi, WiFi SSID, WiFi PSK, режим на WiFi и NTP сървър.                                                                         |
|   [Позиция](./position)   | Опциите за конфигуриране на позицията са: Активиран GPS, Интервал на актуализиране на GPS, GPS Attempt Time, Фиксирана позиция, Smart Broadcast, Интервал на излъчване и Position Packet Flags.            |
| [Захранване](./power)     |          The power config options are: Charge Current, Power Saving, Shutdown after losing power, ADC Multiplier Override Wait Bluetooth Interval, Light Sleep Interval and Minimum Wake Interval.         |
|  [Сигурност](./security)  |                        The security config options are: Public Key, Private Key, Admin Key, Is Managed, Serial Console, Debug Logs, and Admin Channel.                                                     |
| [Потребител](./user)      |                               Опциите за конфигурация на потребителя са: Дълго име, Кратко име и да;о е лицензиран                                                                                         |
